# Kotlin Commands Cheatsheet

> Kotlin is a modern, concise JVM language.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `kotlinc` | Compile Kotlin source files |
| `kotlinc -include-runtime` | Compile with runtime included in the JAR |
| `kotlinc -d` | Set the output directory or JAR file |
| `kotlinc -cp` | Set the classpath for compilation |
| `kotlinc -jvm-target` | Set the JVM target version |
| `kotlin` | Run a compiled Kotlin program |
| `kotlin -script` | Run a Kotlin script file |
| `kotlinc -script` | Compile and run a Kotlin script |
| `kotlin -J` | Pass JVM options to the runtime |
| `kotlinc (REPL)` | Start the Kotlin REPL |
| `ki` | Start the Kotlin interactive shell |
| `gradle init (Kotlin)` | Initialize a Gradle project with Kotlin |
| `gradle build (Kotlin)` | Build a Kotlin project with Gradle |
| `gradle run (Kotlin)` | Run a Kotlin application with Gradle |
| `gradle compileKotlin` | Compile Kotlin sources |
| `gradle jar (Kotlin)` | Package Kotlin project as a JAR |
| `gradle kotlinMultiplatform` | Build a Kotlin Multiplatform project |
| `gradle jvmTest` | Run JVM-targeted tests |
| `gradle publishToMavenLocal` | Publish to local Maven repository |
| `gradle test (Kotlin)` | Run Kotlin tests |
| `gradle test --tests` | Run specific test classes or methods |
| `gradle koverReport` | Generate code coverage report |
| `gradle dependencies (Kotlin)` | Show project dependency tree |
| `gradle dependencyUpdates` | Check for dependency updates |
| `ktlint` | Lint Kotlin code for style issues |
| `ktlint --format` | Auto-format Kotlin code |
| `detekt` | Static code analysis for Kotlin |
| `kotlinc-native` | Compile Kotlin to native binary |
| `gradle linkDebugExecutableNative` | Build debug native executable |
| `gradle linkReleaseExecutableNative` | Build release native executable |
| `gradle jsBrowserRun` | Run Kotlin/JS in a browser |
| `gradle jsNodeRun` | Run Kotlin/JS on Node.js |
| `gradle jsBrowserProductionWebpack` | Build production Kotlin/JS bundle |
