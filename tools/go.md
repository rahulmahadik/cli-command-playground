# Go Commands Cheatsheet

> Go is a compiled, statically typed language from Google. Build fast, efficient programs with clean syntax.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `go build` | Compile packages and dependencies |
| `go run` | Compile and run a Go program |
| `go install` | Compile and install packages |
| `go clean` | Remove object files and cached data |
| `go mod init` | Initialize a new Go module |
| `go mod tidy` | Add missing and remove unused modules |
| `go mod download` | Download modules to local cache |
| `go mod vendor` | Copy dependencies into vendor directory |
| `go mod verify` | Verify dependencies have expected content |
| `go mod graph` | Print the module dependency graph |
| `go mod edit` | Edit the go.mod file programmatically |
| `go mod why` | Explain why a module is needed |
| `go work` | Manage Go workspace for multi-module dev |
| `go test` | Run package tests |
| `go test -v` | Run tests with verbose output |
| `go test -cover` | Run tests with coverage analysis |
| `go test -bench` | Run benchmark tests |
| `go test -race` | Run tests with race condition detection |
| `go fmt` | Format Go source code |
| `gofmt` | Format Go source code (standalone tool) |
| `go vet` | Report suspicious code constructs |
| `staticcheck` | Run advanced static analysis |
| `go doc` | Show documentation for a package |
| `godoc` | Start a documentation server |
| `go get` | Add or update module dependencies |
| `go list` | List packages or modules |
| `go generate` | Run code generation commands |
| `go tool` | Run a specified Go tool |
| `go tool pprof` | Profile CPU and memory usage |
| `govulncheck` | Check for known vulnerabilities |
| `go version` | Display the Go version |
| `go env` | Print Go environment variables |
| `go env -w` | Set a Go environment variable |
