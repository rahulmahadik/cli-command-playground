# Vercel Commands Cheatsheet

> Vercel is a cloud platform for frontend developers.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `vercel login` | Authenticate with Vercel account |
| `vercel logout` | Log out from Vercel CLI |
| `vercel whoami` | Display current authenticated user |
| `vercel switch` | Switch between Vercel teams |
| `vercel` | Deploy project with default settings |
| `vercel deploy` | Deploy project to Vercel |
| `vercel --prod` | Deploy directly to production |
| `vercel promote` | Promote a deployment to production |
| `vercel rollback` | Rollback to a previous deployment |
| `vercel link` | Link local project to Vercel |
| `vercel list` | List all project deployments |
| `vercel ls` | Short alias to list deployments |
| `vercel inspect` | View deployment details and metadata |
| `vercel remove` | Remove a deployment |
| `vercel env ls` | List environment variables |
| `vercel env add` | Add an environment variable |
| `vercel env pull` | Pull env variables to local file |
| `vercel env rm` | Remove an environment variable |
| `vercel dev` | Start local development server |
| `vercel build` | Build project locally |
| `vercel pull` | Pull project settings from Vercel |
| `vercel logs` | View deployment log output |
| `vercel domains` | Manage project custom domains |
| `vercel certs` | Manage SSL certificates |
| `vercel dns` | Manage DNS records |
| `vercel secrets (deprecated)` | Manage secrets (use env instead) |
