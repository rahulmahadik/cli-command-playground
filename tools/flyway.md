# Flyway Commands Cheatsheet

> Flyway is a database migration tool.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `flyway migrate` | Apply pending database migrations |
| `flyway clean` | Drop all objects in configured schemas |
| `flyway baseline` | Baseline an existing database |
| `flyway undo` | Undo the most recent migration |
| `flyway info` | Display migration status and details |
| `flyway validate` | Validate applied migrations against available |
| `flyway repair` | Repair the schema history table |
