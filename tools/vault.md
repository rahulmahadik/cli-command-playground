# Vault Commands Cheatsheet

> Vault is a secrets management platform by HashiCorp.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `vault server` | Start the Vault server process |
| `vault login` | Authenticate with Vault server |
| `vault status` | Check Vault server seal status |
| `vault auth enable` | Enable an authentication method |
| `vault token` | Manage Vault authentication tokens |
| `vault kv put` | Store a secret key-value pair |
| `vault kv get` | Retrieve a stored secret |
| `vault kv list` | List secrets at a path |
| `vault kv delete` | Delete a secret version |
| `vault kv patch` | Update specific secret fields |
| `vault secrets enable` | Enable a secrets engine |
| `vault operator init` | Initialize a new Vault cluster |
| `vault operator unseal` | Unseal the Vault with key share |
| `vault operator seal` | Seal the Vault server |
| `vault policy` | Manage Vault access policies |
