# NuGet Commands Cheatsheet

> NuGet is the package manager for .NET.

## Quick Reference

> **Preview** â€” This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `nuget install` | Install a NuGet package |
| `nuget install -Version` | Install a specific package version |
| `nuget install -OutputDirectory` | Install to a custom output directory |
| `nuget restore` | Restore all project dependencies |
| `nuget restore -ConfigFile` | Restore using specific config file |
| `nuget restore -PackagesDirectory` | Restore to custom packages directory |
| `nuget push` | Push a package to a feed |
| `nuget push -Source` | Push to a specific package source |
| `nuget push -ApiKey` | Push with API key authentication |
| `nuget pack` | Create a NuGet package from project |
| `nuget pack -Version` | Create package with specific version |
| `nuget pack -Properties` | Create package with custom properties |
| `dotnet add package` | Add a NuGet package via dotnet CLI |
| `dotnet add package --version` | Add specific package version via dotnet |
| `dotnet remove package` | Remove a package via dotnet CLI |
| `dotnet list package` | List project package references |
| `dotnet restore` | Restore dependencies via dotnet CLI |
| `nuget locals` | Manage local NuGet package caches |
| `nuget sources` | Manage NuGet package sources |
| `nuget delete` | Delete a package from a feed |
| `nuget config` | Get or set NuGet config values |
