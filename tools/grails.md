# Grails Commands Cheatsheet

> Grails is a full-stack web framework built on Groovy and the JVM.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `grails create-app` | Create a new Grails application |
| `grails create-plugin` | Create a new Grails plugin |
| `grails create-profile` | Create a custom application profile |
| `grails run-app` | Run the application in development mode |
| `grails run-app --port` | Run on a specific port |
| `grails run-app --https` | Run with HTTPS enabled |
| `grails stop-app` | Stop the running application |
| `grails create-domain-class` | Generate a domain class |
| `grails create-controller` | Generate a controller |
| `grails create-service` | Generate a service |
| `grails create-taglib` | Generate a tag library |
| `grails create-interceptor` | Generate an interceptor |
| `grails generate-controller` | Generate a controller from a domain class |
| `grails generate-views` | Generate views from a domain class |
| `grails generate-all` | Generate controller and views from a domain class |
| `grails create-unit-test` | Generate a unit test |
| `grails create-integration-test` | Generate an integration test |
| `grails test-app` | Run all tests |
| `grails test-app --unit` | Run unit tests only |
| `grails test-app --integration` | Run integration tests only |
| `grails test-app ClassName` | Run tests for a specific class |
| `grails war` | Package the app as a WAR file |
| `grails package` | Package the application |
| `grails clean` | Clean build artifacts |
| `grails compile` | Compile the application |
| `grails schema-export` | Export the database schema |
| `grails dbm-update` | Apply pending database migrations |
| `grails dbm-rollback` | Roll back database migrations |
| `grails dbm-gorm-diff` | Generate a migration diff from GORM |
| `grails dbm-generate-changelog` | Generate a changelog from the database |
| `grails dbm-status` | Show migration status |
| `grails dependency-report` | Show project dependencies |
| `grails url-mappings-report` | Show URL mappings |
| `grails list-profiles` | List available application profiles |
| `grails plugin-info` | Show plugin information |
| `grails list-plugins` | List available plugins |
| `grails console` | Open the Grails interactive console |
| `grails shell` | Open the Grails shell |
| `grails interactive` | Start interactive mode |
| `grails run-app -Dgrails.env` | Run with a specific environment |
| `grails war -Dgrails.env` | Package for a specific environment |
| `grails bug-report` | Generate a bug report |
| `grails stats` | Show project statistics |
