# GitHub Copilot Commands Cheatsheet

> GitHub Copilot is an AI-powered coding assistant. Get intelligent command suggestions in the CLI.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| `/newNotebook` | Create a new Jupyter notebook |
| `/help` | Show Copilot help information |
| `/fixTestFailure` | Fix a failing test |
| `/setupTests` | Set up testing framework |
| `/search` | Search the codebase |
| `/startDebugging` | Start a debugging session |
| `@workspace` | Reference the entire workspace |
| `@vscode` | Ask about VS Code features |

| `@github` | Query GitHub data and repos |

| `#usages` | Reference symbol usages |

| `Esc` | Dismiss an inline suggestion |
| `Alt+]` | Show the next suggestion |
| `Alt+[` | Show the previous suggestion |
| `Alt+\` | Trigger an inline suggestion |
| `Ctrl+Enter` | Open completions panel |
| `Ctrl+I` | Open inline chat |
| `Ctrl+Alt+I` | Open the chat panel |
