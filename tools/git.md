# Git Commands Cheatsheet

> Git is a distributed version control system. Track code changes, branch freely, and collaborate with your team.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `git init` | Initialize a new Git repository |
| `git clone` | Clone a remote repository locally |
| `git status` | Show working tree status |
| `git add` | Stage changes for commit |
| `git commit` | Record changes to the repository |
| `git diff` | Show changes between commits or working tree |
| `git rm` | Remove files from working tree and index |
| `git mv` | Move or rename a file or directory |
| `git branch` | List, create, or delete branches |
| `git checkout` | Switch branches or restore files |
| `git switch` | Switch to a different branch |
| `git merge` | Join two or more development histories |
| `git rebase` | Reapply commits on top of another base |
| `git remote` | Manage set of tracked repositories |
| `git fetch` | Download objects and refs from remote |
| `git pull` | Fetch and integrate remote changes |
| `git push` | Update remote refs with local commits |
| `git log` | Show commit history logs |
| `git show` | Display commit details and content |
| `git blame` | Show who last modified each line |
| `git reflog` | Manage reference log entries |
| `git stash` | Stash changes in a dirty working directory |
| `git reset` | Reset current HEAD to a specified state |
| `git revert` | Create a commit that undoes previous changes |
| `git restore` | Restore working tree files |
| `git clean` | Remove untracked files from working tree |
| `git tag` | Create, list, or verify tag objects |
| `git config` | Get and set repository or global options |
| `git cherry-pick` | Apply changes from specific commits |
| `git bisect` | Binary search to find a bad commit |
| `git worktree` | Manage multiple working trees |
