# Apache Kafka Commands Cheatsheet

> Apache Kafka is a distributed event streaming platform.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `kafka-topics.sh --create` | Create a new Kafka topic |
| `kafka-topics.sh --list` | List all available Kafka topics |
| `kafka-topics.sh --describe` | Show topic configuration details |
| `kafka-topics.sh --alter` | Modify topic partition count or config |
| `kafka-topics.sh --delete` | Delete a Kafka topic |
| `kafka-console-producer.sh` | Start console producer to send messages |
| `kafka-console-consumer.sh` | Start console consumer to read messages |
| `kafka-consumer-groups.sh --list` | List all consumer groups |
| `kafka-consumer-groups.sh --describe` | Describe consumer group details and lag |
| `kafka-consumer-groups.sh --reset-offsets` | Reset consumer group offsets |
| `kafka-consumer-groups.sh --delete` | Delete a consumer group |
| `kafka-configs.sh --describe` | Describe broker or topic configurations |
| `kafka-configs.sh --alter` | Modify broker or topic configurations |
| `kafka-acls.sh --add` | Add access control list entries |
| `kafka-acls.sh --list` | List access control list entries |
| `kafka-broker-api-versions.sh` | Display broker API version information |
| `kafka-log-dirs.sh` | Describe log directory usage |
| `kafka-reassign-partitions.sh --execute` | Execute partition reassignment plan |
