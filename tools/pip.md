# pip Commands Cheatsheet

> pip is the package installer for Python. Install and manage Python libraries quickly.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `pip install` | Install a Python package |
| `pip install -r` | Install packages from requirements file |
| `pip install -e` | Install a package in editable mode |
| `pip install --upgrade` | Upgrade an installed package |
| `pip install --user` | Install for the current user only |
| `pip uninstall` | Uninstall a Python package |
| `pip uninstall -y` | Uninstall without confirmation prompt |
| `pip show` | Show information about a package |
| `pip list` | List all installed packages |
| `pip list --outdated` | List packages with updates available |
| `pip check` | Check for dependency conflicts |
| `pip --version` | Display the pip version |
| `pip install --upgrade pip` | Upgrade pip itself to latest version |
| `pip freeze` | Output installed packages in requirements format |
| `pip freeze > requirements.txt` | Save dependencies to a file |
| `pip cache purge` | Remove all pip cache files |
| `pip cache info` | Show pip cache directory info |
| `pip cache list` | List packages in the pip cache |
| `pip index versions` | List available versions of a package |
| `pip config list` | Show pip configuration settings |
| `pip config set` | Set a pip configuration value |
| `pip hash` | Compute hash of a package archive |
| `python -m venv` | Create a virtual environment |
| `source venv/bin/activate` | Activate a virtual environment |
| `deactivate` | Deactivate the current virtual environment |
| `pip download` | Download packages without installing |
| `pip wheel` | Build wheel packages from requirements |
