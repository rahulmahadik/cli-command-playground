# Windows CMD Commands Cheatsheet

> Windows CMD is the command-line interface for Microsoft Windows. Navigate files, configure the system.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `dir` | List directory contents |
| `cd` | Change the current directory |
| `mkdir` | Create a new directory |
| `rmdir` | Remove a directory |
| `copy` | Copy one or more files |
| `xcopy` | Copy files and directory trees |
| `robocopy` | Robust file and directory copy |
| `move` | Move files or directories |
| `del` | Delete one or more files |
| `ren` | Rename a file or directory |
| `attrib` | Display or change file attributes |
| `tree` | Display directory tree graphically |
| `type` | Display the contents of a text file |
| `mklink` | Create symbolic or hard links |
| `where` | Locate a program or file |
| `ipconfig` | Display IP configuration details |
| `ping` | Test network connectivity |
| `tracert` | Trace the route to a host |
| `netstat` | Display network connections and stats |
| `nslookup` | Query DNS name servers |
| `netsh` | Configure network settings |
| `arp` | Display or modify the ARP table |
| `hostname` | Display the computer name |
| `nbtstat` | Display NetBIOS over TCP/IP statistics |
| `route` | Display or modify the routing table |
| `pathping` | Trace route with latency and loss stats |
| `getmac` | Display MAC addresses |
| `net use` | Map or disconnect network drives |
| `net view` | Display shared network resources |
| `curl` | Transfer data from or to a server |
| `systeminfo` | Display detailed system information |
| `ver` | Display the Windows version |
| `wmic` | Windows Management Instrumentation CLI |
| `msinfo32` | Open System Information GUI |
| `driverquery` | List installed device drivers |
| `set` | Display or set environment variables |
| `path` | Display or set the PATH variable |
| `date` | Display or set the system date |
| `time` | Display or set the system time |
| `whoami` | Display the current username |
| `winver` | Show Windows version dialog |
| `tasklist` | List running processes |
| `taskkill` | Kill a running process |
| `start` | Start a program or open a file |
| `schtasks` | Schedule tasks to run automatically |
| `at` | Schedule commands at a specific time |
| `timeout` | Wait for a specified time |
| `waitfor` | Wait for or send a signal |
| `diskpart` | Manage disk partitions |
| `chkdsk` | Check disk for errors |
| `format` | Format a disk for use |
| `label` | Create or change a disk label |
| `vol` | Display the disk volume label |
| `defrag` | Defragment a disk drive |
| `fsutil` | File system utility commands |
| `compact` | Compress or uncompress files on NTFS |
| `cipher` | Encrypt or decrypt NTFS files |
| `cleanmgr` | Open Disk Cleanup utility |
| `net user` | Manage user accounts |
| `net localgroup` | Manage local groups |
| `runas` | Run a program as another user |
| `icacls` | Display or modify file permissions |
| `takeown` | Take ownership of a file |
| `cacls` | Display or change file ACLs |
| `gpupdate` | Refresh Group Policy settings |
| `gpresult` | Display Group Policy results |
| `secedit` | Configure and analyze security |
| `sc query` | Query the status of a service |
| `sc start` | Start a Windows service |
| `sc stop` | Stop a Windows service |
| `sc config` | Configure a service startup type |
| `net start` | Start a Windows service |
| `net stop` | Stop a Windows service |
| `sc create` | Create a new Windows service |
| `sc delete` | Delete a Windows service |
| `echo` | Display messages or toggle echo |
| `if` | Perform conditional processing |
| `for` | Run a command for each item |
| `call` | Call a batch file from another |
| `goto` | Jump to a label in a batch file |
| `exit` | Exit the command interpreter |
| `pause` | Pause batch file execution |
| `rem` | Add a comment in a batch file |
| `cls` | Clear the console screen |
| `title` | Set the console window title |
| `color` | Set console text and background color |
| `prompt` | Change the command prompt format |
| `setlocal` | Begin localization of variables |
| `endlocal` | End localization of variables |
| `reg query` | Read a registry key value |
| `reg add` | Add a registry key or value |
| `reg delete` | Delete a registry key or value |
| `reg export` | Export a registry key to a file |
| `reg import` | Import a registry file |
| `reg save` | Save a registry hive to a file |
| `reg restore` | Restore a registry hive from file |
| `reg copy` | Copy a registry key |
| `regedit` | Open the Registry Editor GUI |
| `shutdown` | Shut down or restart the computer |
| `powercfg` | Configure power settings |
| `logoff` | Log off the current user |
| `hibernate` | Hibernate the computer |
| `find` | Search for a text string in files |
| `findstr` | Search for strings using regex |
| `sort` | Sort lines of text |
| `more` | Display output one screen at a time |
| `fc` | Compare two files |
| `comp` | Compare contents of two files |
| `clip` | Copy command output to clipboard |
| `sfc` | System File Checker scan and repair |
| `dism` | Service and manage Windows images |
| `bootrec` | Repair the boot configuration |
| `bcdedit` | Edit boot configuration data |
| `eventvwr` | Open the Event Viewer |
| `perfmon` | Open the Performance Monitor |
| `resmon` | Open the Resource Monitor |
| `msconfig` | Open System Configuration utility |
