# Homebrew Commands Cheatsheet

> Homebrew is the missing package manager for macOS (and Linux).

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `brew install` | Install a formula |
| `brew install --cask` | Install a cask (GUI application) |
| `brew reinstall` | Reinstall a formula |
| `brew fetch` | Download a formula without installing |
| `brew uninstall` | Uninstall a formula |
| `brew autoremove` | Remove unused dependencies |
| `brew cleanup` | Remove old versions and cache files |
| `brew search` | Search for available formulae |
| `brew info` | Show information about a formula |
| `brew list` | List installed formulae |
| `brew deps` | Show dependencies for a formula |
| `brew leaves` | List installed formulae with no dependents |
| `brew update` | Update Homebrew and formulae list |
| `brew upgrade` | Upgrade all installed formulae |
| `brew outdated` | List formulae with available updates |
| `brew pin` | Pin a formula to prevent upgrades |
| `brew unpin` | Unpin a formula to allow upgrades |
| `brew doctor` | Diagnose common Homebrew issues |
| `brew config` | Show Homebrew configuration |
| `brew --prefix` | Show Homebrew installation path |
| `brew --cellar` | Show Homebrew Cellar path |
| `brew list --cask` | List installed casks |
| `brew outdated --cask` | List outdated casks |
| `brew upgrade --cask` | Upgrade all casks |
| `brew uninstall --cask` | Uninstall a cask |
| `brew tap` | Add a third-party repository |
| `brew untap` | Remove a third-party repository |
| `brew services list` | List managed background services |
| `brew services start` | Start a background service |
| `brew services stop` | Stop a background service |
