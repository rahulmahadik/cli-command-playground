# GraalVM Commands Cheatsheet

> GraalVM is a high-performance polyglot virtual machine from Oracle.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `native-image` | Build a native executable from Java bytecode |
| `native-image-configure` | Generate native image configuration files |
| `gu install` | Install a GraalVM component |
| `gu list` | List installed GraalVM components |
| `gu available` | List available GraalVM components |
| `gu remove` | Remove an installed component |
| `js` | Launch the GraalVM JavaScript engine |
| `node` | Run Node.js on GraalVM |
| `native-image --pgo-instrument` | Build with profile-guided optimization instrumentation |
