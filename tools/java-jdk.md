# Java JDK Commands Cheatsheet

> Java JDK is the development kit for the Java programming language. Compile, run, and manage Java apps.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `javac` | Compile Java source files to bytecode |
| `javadoc` | Generate API documentation from source |
| `javap` | Disassemble Java class files |
| `java` | Run a compiled Java application |
| `jshell` | Launch the interactive Java REPL |
| `jar` | Create and manage Java archive files |
| `jps` | List running Java processes |
| `keytool` | Manage cryptographic keys and certificates |
| `jstack` | Print Java thread stack traces |
| `jcmd` | Send diagnostic commands to a JVM |
