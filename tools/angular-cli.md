# Angular CLI Commands Cheatsheet

> Angular CLI (ng) is the official command-line tool for the Angular framework.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `ng new` | Create a new Angular workspace |
| `ng new --standalone` | Create a standalone Angular application |
| `ng generate component` | Generate a new component |
| `ng generate service` | Generate a new service |
| `ng generate module` | Generate a new module |
| `ng generate directive` | Generate a new directive |
| `ng generate pipe` | Generate a new pipe |
| `ng generate guard` | Generate a route guard |
| `ng generate interceptor` | Generate an HTTP interceptor |
| `ng serve` | Start the development server |
| `ng serve --port` | Serve on a specific port |
| `ng serve --open` | Serve and open in the browser |
| `ng build` | Build the application |
| `ng test` | Run unit tests via Karma |
| `ng e2e` | Run end-to-end tests |
| `ng lint` | Lint the project code |
| `ng add` | Add a library to the project |
| `ng update` | Update Angular and dependencies |
| `ng config` | View or set Angular CLI configuration |
| `ng analytics` | Configure CLI analytics |
| `ng version` | Display Angular CLI version |
| `ng doc` | Open Angular documentation |
| `ng completion` | Set up CLI autocompletion |
