# Mercurial Commands Cheatsheet

> Mercurial is a distributed version control system.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `hg init` | Create a new repository |
| `hg clone` | Clone an existing repository |
| `hg status` | Show working directory status |
| `hg add` | Add files to tracking |
| `hg remove` | Remove files from tracking |
| `hg rename` | Rename a file |
| `hg copy` | Copy a file |
| `hg commit` | Commit changes to the repository |
| `hg diff` | Show differences in working directory |
| `hg addremove` | Add new files and remove missing files |
| `hg branch` | Set or show the current branch name |
| `hg branches` | List repository branches |
| `hg merge` | Merge another branch into the current one |
| `hg update` | Update the working directory |
| `hg bookmark` | Create or manage bookmarks |
| `hg push` | Push changes to a remote repository |
| `hg pull` | Pull changes from a remote repository |
| `hg incoming` | Show incoming changesets |
| `hg outgoing` | Show outgoing changesets |
| `hg paths` | Show configured repository paths |
| `hg log` | Show revision history |
| `hg annotate` | Show per-line changeset information |
| `hg cat` | Output file contents at a revision |
| `hg summary` | Show a summary of the working directory |
| `hg revert` | Revert files to a previous state |
| `hg backout` | Create a changeset to reverse an earlier one |
| `hg rollback` | Undo the last transaction |
| `hg strip` | Remove changesets from the repository |
| `hg config` | Show or edit configuration |
| `hg graft` | Copy changesets from another branch |
| `hg rebase` | Move changesets to a new base |
| `hg histedit` | Interactively edit changeset history |
| `hg bisect` | Binary search for a changeset |
| `hg help` | Show help for a command |
| `hg version` | Display Mercurial version |
| `hg serve` | Start a web server for the repository |
| `hg archive` | Create an archive of the repository |
| `hg tag` | Add a tag to a changeset |
