# OpenShift Commands Cheatsheet

> OpenShift CLI (oc) is Red Hat's command-line tool for OpenShift Kubernetes.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `oc login` | Log in to an OpenShift cluster |
| `oc logout` | Log out of the cluster |
| `oc whoami` | Display the current user |
| `oc status` | Show the current project status |
| `oc new-project` | Create a new project |
| `oc project` | Switch to a project |
| `oc projects` | List available projects |
| `oc delete project` | Delete a project |
| `oc new-app` | Create a new application |
| `oc expose` | Expose a service as a route |
| `oc rollout` | Manage application rollouts |
| `oc set image` | Update container images in a deployment |
| `oc rollback` | Roll back to a previous deployment |
| `oc get` | List resources |
| `oc describe` | Show resource details |
| `oc create` | Create a resource from a file |
| `oc apply` | Apply configuration to a resource |
| `oc delete` | Delete resources |
| `oc edit` | Edit a resource in place |
| `oc logs` | View container logs |
| `oc rsh` | Open a remote shell in a pod |
| `oc exec` | Execute a command in a container |
| `oc debug` | Start a debug pod for troubleshooting |
| `oc port-forward` | Forward local ports to a pod |
| `oc scale` | Scale a deployment |
| `oc autoscale` | Configure autoscaling |
| `oc set env` | Set environment variables |
| `oc set resources` | Set resource requests and limits |
| `oc start-build` | Start a build |
| `oc import-image` | Import an image into the registry |
| `oc tag` | Tag an image |
| `oc new-build` | Create a new build configuration |
| `oc adm policy` | Manage security policies |
| `oc create secret` | Create a secret |
| `oc sa` | Manage service accounts |
| `oc adm groups` | Manage user groups |
