# Cargo (Rust) Commands Cheatsheet

> Cargo is the package manager and build tool for Rust.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `cargo new` | Create a new Rust project |
| `cargo init` | Initialize Rust project in current directory |
| `cargo build` | Compile the current project |
| `cargo run` | Build and run the current project |
| `cargo check` | Check code for errors without building |
| `cargo clean` | Remove build artifacts directory |
| `cargo watch` | Watch for changes and rebuild automatically |
| `cargo test` | Run all tests in the project |
| `cargo bench` | Run benchmark tests for performance |
| `cargo add` | Add a dependency to Cargo.toml |
| `cargo remove` | Remove a dependency from Cargo.toml |
| `cargo update` | Update dependencies to latest versions |
| `cargo tree` | Display dependency tree structure |
| `cargo vendor` | Vendor all dependencies locally |
| `cargo fmt` | Format source code with rustfmt |
| `cargo clippy` | Run linter for common mistakes |
| `cargo fix` | Automatically fix compiler warnings |
| `cargo doc` | Generate documentation for the project |
| `cargo publish` | Publish package to crates.io registry |
| `cargo login` | Authenticate with crates.io API token |
| `cargo package` | Package project into distributable crate |
| `cargo install` | Install a Rust binary globally |
| `cargo search` | Search for crates on crates.io |
| `cargo --version` | Display installed Cargo version |
