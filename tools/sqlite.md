# SQLite Commands Cheatsheet

> SQLite is a lightweight, embedded SQL database engine.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `sqlite3` | Open the SQLite shell |
| `.open` | Open or create a database file |
| `.databases` | List attached databases |
| `.quit` | Exit the SQLite shell |
| `CREATE TABLE` | Create a new table |
| `.tables` | List all tables |
| `.schema` | Show the schema for a table |
| `DROP TABLE` | Delete a table |
| `ALTER TABLE` | Modify a table structure |
| `CREATE INDEX` | Create an index on a table |
| `SELECT` | Query data from a table |
| `WHERE` | Filter query results |
| `JOIN` | Combine data from multiple tables |
| `GROUP BY` | Group results by a column |
| `ORDER BY` | Sort query results |
| `LIMIT` | Limit the number of results |
| `INSERT INTO` | Insert new data |
| `UPDATE` | Update existing data |
| `DELETE` | Delete data from a table |
| `REPLACE` | Insert or replace data |
| `.mode` | Set the output mode (csv, json, table, etc.) |
| `.headers` | Toggle column headers on/off |
| `.output` | Redirect output to a file |
| `.read` | Execute SQL from a file |
| `.show` | Show current settings |
| `.backup` | Back up a database to a file |
| `.restore` | Restore a database from a file |
| `.timer` | Toggle SQL execution timer |
| `VACUUM` | Rebuild and compact the database |
| `ATTACH DATABASE` | Attach another database file |
| `.explain` | Show the query execution plan |
| `.import` | Import data from a CSV file |
| `.dump` | Export the database as SQL |
| `.excel` | Open results in a spreadsheet |
