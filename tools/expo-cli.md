# Expo Commands Cheatsheet

> Expo is an open-source platform for React Native development.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `npx create-expo-app` | Create a new Expo project |
| `npx create-expo-app --template` | Create project from a template |
| `npx expo install` | Install compatible Expo packages |
| `npx expo start` | Start Expo development server |
| `npx expo start --android` | Start and open on Android device |
| `npx expo start --ios` | Start and open on iOS simulator |
| `npx expo start --web` | Start and open in web browser |
| `npx expo start --clear` | Start with cleared bundler cache |
| `eas build` | Build app using EAS Build |
| `eas build --platform android` | Build for Android platform |
| `eas build --platform ios` | Build for iOS platform |
| `eas build --profile production` | Build using production profile |
| `eas submit` | Submit app to app stores |
| `eas submit --platform android` | Submit to Google Play Store |
| `eas submit --platform ios` | Submit to Apple App Store |
| `eas update` | Publish an over-the-air update |
| `npx expo prebuild` | Generate native project files |
| `npx expo doctor` | Diagnose project configuration issues |
