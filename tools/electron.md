# Electron Commands Cheatsheet

> Electron is a framework for building cross-platform desktop apps.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `npx create-electron-app` | Create a new Electron application |
| `electron .` | Launch Electron app from current directory |
| `npm start` | Start Electron app via npm script |
| `electron-builder` | Build distributable Electron packages |
| `electron-forge make` | Build platform-specific distributables |
| `electron-forge package` | Package app without creating installer |
| `npx electron --inspect` | Start Electron with debugging enabled |
