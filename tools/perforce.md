# Perforce Commands Cheatsheet

> Perforce is an enterprise version control platform.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `p4 client` | Create or edit a client workspace |
| `p4 set` | Set or view Perforce environment variables |
| `p4 info` | Display client and server information |
| `p4 login` | Authenticate with the Perforce server |
| `p4 add` | Open a file for add |
| `p4 edit` | Open a file for edit |
| `p4 delete` | Open a file for delete |
| `p4 move` | Move or rename a file |
| `p4 revert` | Discard changes to open files |
| `p4 change` | Create or edit a changelist |
| `p4 submit` | Submit a changelist to the depot |
| `p4 changes` | List submitted changelists |
| `p4 opened` | List currently open files |
| `p4 sync` | Synchronize files from the depot |
| `p4 flush` | Update the have list without syncing |
| `p4 integrate` | Integrate changes between branches |
| `p4 merge` | Merge changes from another branch |
| `p4 resolve` | Resolve file conflicts |
| `p4 copy` | Copy files between branches |
| `p4 filelog` | Show file revision history |
| `p4 diff` | Diff a client file against the depot |
| `p4 diff2` | Diff two depot files |
| `p4 annotate` | Show per-line file annotation |
| `p4 describe` | Describe a changelist |
| `p4 shelve` | Shelve files for later use |
| `p4 unshelve` | Restore shelved files |
| `p4 depot` | Create or edit a depot |
| `p4 user` | Create or edit a user specification |
| `p4 group` | Create or edit a user group |
| `p4 protect` | Edit the protection table |
