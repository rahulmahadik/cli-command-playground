# Consul Commands Cheatsheet

> Consul is a service mesh and service discovery tool by HashiCorp.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `consul agent` | Start a Consul agent process |
| `consul members` | List all cluster member nodes |
| `consul info` | Display agent configuration information |
| `consul services register` | Register a service with Consul |
| `consul services deregister` | Deregister a service from Consul |
| `consul catalog services` | List all registered services |
| `consul catalog nodes` | List all known cluster nodes |
| `consul kv put` | Store a key-value pair |
| `consul kv get` | Retrieve a value by key |
| `consul kv delete` | Delete a key-value entry |
| `consul kv export` | Export key-value store data |
| `consul kv import` | Import key-value store data |
| `consul connect proxy` | Start a Connect sidecar proxy |
| `consul connect ca` | Manage Connect certificate authority |
| `consul intention create` | Create a service-to-service intention |
| `consul intention list` | List all service intentions |
| `consul intention delete` | Delete a service intention |
| `consul acl bootstrap` | Bootstrap the ACL system |
| `consul acl token create` | Create a new ACL token |
| `consul acl token list` | List all ACL tokens |
| `consul acl policy create` | Create a new ACL policy |
| `consul acl policy read` | Read an ACL policy |
| `consul join` | Join agent to a cluster |
| `consul leave` | Gracefully leave the cluster |
| `consul reload` | Reload agent configuration |
| `consul snapshot save` | Save cluster state snapshot |
| `consul snapshot restore` | Restore cluster from snapshot |
