# GitHub CLI Commands Cheatsheet

> GitHub CLI is the official command-line tool for GitHub. Manage repos, open PRs, review code.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `gh auth login` | Authenticate with GitHub |
| `gh auth status` | Show authentication status |
| `gh auth logout` | Log out of GitHub |
| `gh auth refresh` | Refresh stored credentials |
| `gh auth token` | Print the authentication token |
| `gh auth setup-git` | Configure git to use GitHub CLI |
| `gh repo clone` | Clone a GitHub repository |
| `gh repo create` | Create a new GitHub repository |
| `gh repo view` | View repository details |
| `gh repo list` | List your repositories |
| `gh repo fork` | Fork a repository |
| `gh repo delete` | Delete a repository |
| `gh repo rename` | Rename a repository |
| `gh repo sync` | Sync a forked repository |
| `gh pr create` | Create a new pull request |
| `gh pr list` | List open pull requests |
| `gh pr view` | View pull request details |
| `gh pr checkout` | Check out a pull request branch |
| `gh pr merge` | Merge a pull request |
| `gh pr review` | Review a pull request |
| `gh pr close` | Close a pull request |
| `gh pr diff` | View pull request diff |
| `gh pr status` | Show pull request status |
| `gh pr edit` | Edit a pull request |
| `gh pr checks` | View pull request check results |
| `gh issue create` | Create a new issue |
| `gh issue list` | List repository issues |
| `gh issue view` | View issue details |
| `gh issue close` | Close an issue |
| `gh issue comment` | Comment on an issue |
| `gh issue edit` | Edit an issue |
| `gh issue reopen` | Reopen a closed issue |
| `gh workflow list` | List repository workflows |
| `gh workflow run` | Trigger a workflow run |
| `gh workflow view` | View workflow details |
| `gh run list` | List recent workflow runs |
| `gh run view` | View workflow run details |
| `gh run watch` | Watch a workflow run live |
| `gh run rerun` | Re-run a failed workflow |
| `gh run cancel` | Cancel a workflow run |
| `gh release create` | Create a new release |
| `gh release list` | List repository releases |
| `gh release view` | View release details |
| `gh release download` | Download release assets |
| `gh release delete` | Delete a release |
| `gh release upload` | Upload assets to a release |
| `gh gist create` | Create a new gist |
| `gh gist list` | List your gists |
| `gh gist view` | View a gist |
| `gh gist clone` | Clone a gist locally |
| `gh gist delete` | Delete a gist |
| `gh api` | Make authenticated GitHub API calls |
| `gh extension install` | Install a CLI extension |
| `gh extension list` | List installed extensions |
| `gh config set` | Set a configuration value |
| `gh alias set` | Create a command alias |
| `gh search repos` | Search for GitHub repositories |
| `gh secret set` | Set a repository secret |
| `gh label list` | List repository labels |
