# Flutter Commands Cheatsheet

> Flutter is Google's open-source UI toolkit for cross-platform development.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `flutter create` | Create a new Flutter project |
| `flutter create --platforms` | Create project for specific platforms |
| `flutter create --template` | Create project from a template |
| `flutter pub get` | Install project dependencies |
| `flutter pub add` | Add a package dependency |
| `flutter clean` | Delete build and temporary files |
| `flutter run` | Run app on connected device |
| `flutter run --release` | Run app in release mode |
| `flutter run -d` | Run app on a specific device |
| `flutter devices` | List all connected devices |
| `flutter emulators` | List available emulators |
| `flutter attach` | Attach to a running Flutter app |
| `flutter build` | Build the Flutter application |
| `flutter build apk` | Build Android APK file |
| `flutter build appbundle` | Build Android App Bundle |
| `flutter build ios` | Build iOS application archive |
| `flutter build web` | Build web application output |
| `flutter build windows` | Build Windows desktop application |
| `flutter test` | Run all unit and widget tests |
| `flutter test --coverage` | Run tests with code coverage |
| `flutter drive` | Run integration and driver tests |
| `flutter analyze` | Analyze Dart code for issues |
| `flutter doctor` | Check Flutter environment setup |
| `flutter doctor -v` | Show verbose environment diagnostics |
| `flutter upgrade` | Upgrade Flutter to latest version |
| `flutter channel` | List or switch Flutter channels |
| `flutter config` | Configure Flutter settings |
