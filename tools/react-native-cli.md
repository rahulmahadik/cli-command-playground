# React Native Commands Cheatsheet

> React Native is a framework for building native mobile apps with React.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `npx react-native init` | Create a new React Native project |
| `npx react-native init --template` | Create project from custom template |
| `npx react-native run-android` | Build and run on Android device |
| `npx react-native run-android --variant` | Run with a specific build variant |
| `npx react-native run-android --deviceId` | Run on a specific Android device |
| `npx react-native run-ios` | Build and run on iOS simulator |
| `npx react-native run-ios --simulator` | Run on specific iOS simulator |
| `npx react-native run-ios --device` | Run on a physical iOS device |
| `npx react-native log-android` | Show Android device log output |
| `npx react-native log-ios` | Show iOS device log output |
| `./gradlew bundleRelease` | Build Android release app bundle |
| `xcodebuild (iOS Release)` | Build iOS release application |
| `npx react-native start` | Start Metro bundler server |
| `npx react-native start --reset-cache` | Start Metro with cleared cache |
| `npx react-native start --port` | Start Metro on specific port |
| `npx react-native doctor` | Check development environment setup |
| `npx react-native info` | Display environment and project info |
| `npx react-native upgrade` | Upgrade React Native version |
| `npx react-native clean` | Clean build artifacts and caches |
