# Playwright Commands Cheatsheet

> Playwright is a browser automation and testing library from Microsoft.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `npm init playwright` | Initialize Playwright in a project |
| `npx playwright install` | Install browser binaries |
| `npx playwright install-deps` | Install system dependencies for browsers |
| `npx playwright test` | Run all Playwright tests |
| `npx playwright test --headed` | Run tests in headed browser mode |
| `npx playwright test --project` | Run tests for specific browser project |
| `npx playwright test --grep` | Run tests matching a pattern |
| `npx playwright test --update-snapshots` | Update visual comparison snapshots |
| `npx playwright codegen` | Generate test code by recording actions |
| `npx playwright test --debug` | Run tests in debug mode |
| `npx playwright show-report` | Open HTML test report |
| `npx playwright show-trace` | View recorded trace file |
| `npx playwright test --ui` | Run tests with interactive UI mode |
| `npx playwright codegen --save-storage` | Record and save authentication state |
| `npx playwright codegen --device` | Record with device emulation |
| `npx playwright test --workers` | Set number of parallel workers |
| `npx playwright test --reporter` | Specify test result reporter |
| `npx playwright test --retries` | Set number of test retries |
| `npx playwright test --list` | List all available tests |
| `npx playwright --version` | Display installed Playwright version |
| `npx playwright open` | Open browser for manual testing |
| `npx playwright test --trace` | Run tests with trace recording |
| `npx playwright test --last-failed` | Re-run only previously failed tests |
