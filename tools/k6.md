# k6 Commands Cheatsheet

> k6 is an open-source load testing tool.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `k6 run` | Run a load test script |
| `k6 run --vus --duration` | Run with specified virtual users and duration |
| `k6 run --iterations` | Run for a fixed number of iterations |
| `k6 run --env` | Pass environment variables to a test |
| `k6 run --stage` | Define ramp-up/ramp-down stages |
| `k6 cloud` | Run a test on the k6 Cloud |
| `k6 login cloud` | Authenticate with k6 Cloud |
| `k6 new` | Create a new k6 test script |
| `k6 inspect` | Inspect a test script without running |
| `k6 archive` | Create a test archive bundle |
| `k6 version` | Display k6 version |
| `k6 run --out` | Send results to an output backend |
| `k6 run with thresholds` | Run with pass/fail threshold criteria |
| `k6 run --summary-export` | Export test summary to a JSON file |
| `k6 scenarios` | Configure multiple test scenarios |
| `k6 executors` | Define test execution models |
| `k6 checks` | Add pass/fail assertions to tests |
