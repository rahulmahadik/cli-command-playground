# Android SDK Commands Cheatsheet

> Android SDK is Google's development toolkit for Android applications.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `sdkmanager --list` | List available and installed SDK packages |
| `sdkmanager --install` | Install SDK packages and tools |
| `avdmanager create avd` | Create a new Android virtual device |
| `adb devices` | List connected Android devices |
| `adb install` | Install APK on connected device |
| `adb logcat` | View device log output stream |
| `adb shell` | Open interactive device shell |
| `fastboot flash` | Flash firmware image to device |
| `emulator -avd` | Launch an Android virtual device |
