# Snyk Commands Cheatsheet

> Snyk is a developer-first security platform.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `snyk auth` | Authenticate with Snyk account |
| `snyk config` | Manage Snyk CLI configuration |
| `snyk test` | Test project for vulnerabilities |
| `snyk monitor` | Monitor project for new vulnerabilities |
| `snyk sbom` | Generate software bill of materials |
| `snyk container test` | Test container image for vulnerabilities |
| `snyk container monitor` | Monitor container image continuously |
| `snyk container sbom` | Generate container image SBOM |
| `snyk code test` | Test source code for security issues |
| `snyk code print-bundle` | Print code analysis bundle info |
| `snyk iac test` | Test infrastructure-as-code files |
| `snyk iac describe` | Detect infrastructure drift |
| `snyk iac update-exclude-policy` | Update IaC exclusion policy |
| `snyk iac rules init` | Initialize custom IaC rules project |
| `snyk fix` | Auto-fix vulnerable dependencies |
| `snyk ignore` | Ignore a specific vulnerability |
| `snyk policy` | Display the Snyk policy file |
| `snyk wizard` | Interactive wizard to fix issues |
| `snyk protect` | Apply patches to vulnerable packages |
| `snyk log4shell` | Detect Log4Shell vulnerability |
