# Grafana Commands Cheatsheet

> Grafana is an open-source analytics and monitoring platform.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `grafana-cli --version` | Display installed Grafana CLI version |
| `grafana-cli --help` | Show Grafana CLI help information |
| `grafana-cli plugins install` | Install a Grafana plugin |
| `grafana-cli plugins ls` | List installed Grafana plugins |
| `grafana-cli plugins list-remote` | List available remote plugins |
| `grafana-cli plugins list-versions` | List available plugin versions |
| `grafana-cli plugins update` | Update a specific plugin |
| `grafana-cli plugins update-all` | Update all installed plugins |
| `grafana-cli plugins uninstall` | Uninstall a Grafana plugin |
| `grafana-cli admin reset-admin-password` | Reset the admin user password |
| `grafana-cli admin data-migration` | Run database migration scripts |
| `grafana-cli admin secrets-migration` | Migrate secrets encryption |
| `grafana-server` | Start the Grafana server process |
| `systemctl start grafana-server` | Start Grafana via systemd service |
| `systemctl status grafana-server` | Check Grafana service status |
| `systemctl enable grafana-server` | Enable Grafana service on boot |
