# Apache Ant Commands Cheatsheet

> Apache Ant is a Java build tool.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `ant` | Run the default target in build.xml |
| `ant -version` | Display the Ant version |
| `ant -help` | Show help information |
| `ant [target]` | Run a specific build target |
| `ant -projecthelp` | List available targets in the build file |
| `ant -keep-going` | Continue building after errors |
| `ant -f` | Use an alternate build file |
| `ant -find` | Search parent directories for a build file |
| `ant -lib` | Add a directory or JAR to the classpath |
| `ant -D` | Define a property (key=value) |
| `ant -propertyfile` | Load properties from a file |
| `ant -v` | Run in verbose mode |
| `ant -q` | Run in quiet mode |
| `ant -l` | Log output to a file |
| `ant -debug` | Run in debug mode |
| `ant -diagnostics` | Show diagnostic information |
