# OpenAPI Generator Commands Cheatsheet

> OpenAPI Generator is a code generation toolkit for API specifications.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `openapi-generator version` | Display the generator version |
| `openapi-generator list` | List available generators |
| `openapi-generator help` | Show help information |
| `openapi-generator validate` | Validate an OpenAPI specification file |
| `openapi-generator config-help` | Show configuration options for a generator |
| `openapi-generator generate` | Generate client/server code from a spec |
| `openapi-generator generate typescript` | Generate TypeScript client code |
| `openapi-generator generate python` | Generate Python client code |
| `openapi-generator generate java` | Generate Java client code |
| `openapi-generator generate spring` | Generate Spring server code |
| `openapi-generator generate go` | Generate Go client code |
| `openapi-generator generate nodejs` | Generate Node.js server code |
| `openapi-generator generate html` | Generate HTML documentation |
| `openapi-generator generate markdown` | Generate Markdown documentation |
| `openapi-generator generate --dry-run` | Preview generation without writing files |
| `openapi-generator author template` | Create custom generator templates |
| `openapi-generator meta` | Generate a new generator module |
| `openapi-generator batch` | Run batch generation from a config file |
