# WP-CLI Commands Cheatsheet

> WP-CLI is the official command-line interface for WordPress.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `wp core download` | Download WordPress core files |
| `wp core install` | Install WordPress |
| `wp core update` | Update WordPress to the latest version |
| `wp core version` | Display the WordPress version |
| `wp core check-update` | Check for available updates |
| `wp plugin install` | Install a plugin |
| `wp plugin activate` | Activate a plugin |
| `wp plugin deactivate` | Deactivate a plugin |
| `wp plugin update` | Update a plugin |
| `wp plugin delete` | Delete a plugin |
| `wp plugin list` | List installed plugins |
| `wp plugin search` | Search the plugin directory |
| `wp theme install` | Install a theme |
| `wp theme activate` | Activate a theme |
| `wp theme update` | Update a theme |
| `wp theme delete` | Delete a theme |
| `wp theme list` | List installed themes |
| `wp user create` | Create a new user |
| `wp user list` | List all users |
| `wp user delete` | Delete a user |
| `wp user update` | Update user details |
| `wp user generate` | Generate dummy users |
| `wp db export` | Export the database |
| `wp db import` | Import a database dump |
| `wp db query` | Execute a SQL query |
| `wp db search` | Search the database |
| `wp db optimize` | Optimize database tables |
| `wp db repair` | Repair database tables |
| `wp post create` | Create a new post |
| `wp post list` | List posts |
| `wp post delete` | Delete a post |
| `wp option get` | Get a WordPress option value |
| `wp option update` | Update a WordPress option |
| `wp search-replace` | Search and replace in the database |
| `wp rewrite flush` | Flush rewrite rules |
| `wp cache flush` | Flush the object cache |
| `wp config create` | Create a wp-config.php file |
| `wp config set` | Set a wp-config.php constant |
| `wp config get` | Get a wp-config.php constant value |
| `wp config list` | List wp-config.php constants |
