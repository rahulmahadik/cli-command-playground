# Podman Commands Cheatsheet

> Podman is a daemonless container engine.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `podman run` | Run a container from an image |
| `podman ps` | List running containers |
| `podman exec` | Execute command in running container |
| `podman stop` | Stop a running container |
| `podman start` | Start a stopped container |
| `podman rm` | Remove a stopped container |
| `podman logs` | View container log output |
| `podman inspect` | View detailed container information |
| `podman images` | List locally available images |
| `podman build` | Build image from a Containerfile |
| `podman pull` | Pull image from a registry |
| `podman push` | Push image to a registry |
| `podman rmi` | Remove a local image |
| `podman tag` | Tag an image with new name |
| `podman pod create` | Create a new pod |
| `podman pod ps` | List all pods |
| `podman pod start` | Start a stopped pod |
| `podman pod stop` | Stop a running pod |
| `podman pod rm` | Remove a pod |
| `podman volume create` | Create a named storage volume |
| `podman volume ls` | List all storage volumes |
| `podman network create` | Create a container network |
| `podman network ls` | List container networks |
| `podman compose up` | Start services from compose file |
| `podman system prune` | Remove unused containers and images |
| `podman login` | Log in to a container registry |
| `podman info` | Display system-wide Podman information |
| `podman generate kube` | Generate Kubernetes YAML from container |
| `podman play kube` | Create containers from Kubernetes YAML |
| `podman generate systemd` | Generate systemd unit files |
| `podman machine init` | Initialize a Podman virtual machine |
| `podman machine start` | Start the Podman virtual machine |
| `podman machine stop` | Stop the Podman virtual machine |
