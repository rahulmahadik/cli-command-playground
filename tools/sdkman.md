# SDKMAN Commands Cheatsheet

> SDKMAN is the Software Development Kit Manager for the JVM.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `sdk install` | Install a specific SDK version |
| `sdk uninstall` | Uninstall a specific SDK version |
| `sdk list` | List all available SDKs |
| `sdk list java` | List available Java versions |
| `sdk current` | Show the currently active SDK version |
| `sdk version` | Display the SDKMAN version |
| `sdk use` | Switch to a specific SDK version for the session |
| `sdk default` | Set the default SDK version |
| `sdk env` | Switch to SDK versions defined in .sdkmanrc |
| `sdk env init` | Create a .sdkmanrc file |
| `sdk upgrade` | Upgrade an SDK to the latest version |
| `sdk config` | Edit SDKMAN configuration |
| `sdk selfupdate` | Update SDKMAN itself |
| `sdk update` | Refresh the list of available SDKs |
| `sdk flush` | Clear local caches |
| `sdk home` | Show the installation path for an SDK |
| `sdk offline` | Toggle offline mode |
| `sdk broadcast` | Show the latest SDK announcements |
