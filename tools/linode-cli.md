# Linode CLI Commands Cheatsheet

> Linode CLI is the official command-line tool for Akamai Cloud (Linode).

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `linode-cli configure` | Configure authentication |
| `linode-cli --version` | Display the CLI version |
| `linode-cli regions list` | List available regions |
| `linode-cli linodes create` | Create a new Linode instance |
| `linode-cli linodes list` | List all Linode instances |
| `linode-cli linodes view` | View details of a Linode |
| `linode-cli linodes boot` | Boot a Linode instance |
| `linode-cli linodes reboot` | Reboot a Linode instance |
| `linode-cli linodes shutdown` | Shut down a Linode instance |
| `linode-cli linodes delete` | Delete a Linode instance |
| `linode-cli linodes resize` | Resize a Linode instance |
| `linode-cli networking ips-list` | List IP addresses |
| `linode-cli networking ip-add` | Add an IP address |
| `linode-cli nodebalancers create` | Create a NodeBalancer |
| `linode-cli nodebalancers list` | List NodeBalancers |
| `linode-cli volumes create` | Create a block storage volume |
| `linode-cli volumes list` | List volumes |
| `linode-cli volumes attach` | Attach a volume to a Linode |
| `linode-cli volumes detach` | Detach a volume |
| `linode-cli volumes delete` | Delete a volume |
| `linode-cli firewalls create` | Create a firewall |
| `linode-cli firewalls list` | List firewalls |
| `linode-cli firewalls rules-update` | Update firewall rules |
| `linode-cli domains create` | Create a domain |
| `linode-cli domains list` | List domains |
| `linode-cli domains records-create` | Create a DNS record |
| `linode-cli lke cluster-create` | Create a Kubernetes cluster |
| `linode-cli lke clusters-list` | List Kubernetes clusters |
| `linode-cli lke pools-list` | List node pools |
| `linode-cli databases mysql-list` | List MySQL databases |
| `linode-cli account view` | View account information |
| `linode-cli events list` | List account events |
| `linode-cli images list` | List available images |
