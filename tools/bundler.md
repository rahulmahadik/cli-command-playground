# Bundler Commands Cheatsheet

> Bundler is a dependency manager for Ruby projects.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `bundle install` | Install all gems from Gemfile |
| `bundle add` | Add a gem to the Gemfile |
| `bundle remove` | Remove a gem from the Gemfile |
| `bundle update` | Update gems to latest versions |
| `bundle outdated` | List gems with newer versions |
| `bundle show` | Show installed gem file path |
| `bundle list` | List all installed gems |
| `bundle info` | Display detailed gem information |
| `bundle open` | Open gem source in editor |
| `bundle init` | Create a new Gemfile |
| `bundle lock` | Generate or update Gemfile.lock |
| `bundle check` | Verify all gems are installed |
| `bundle exec` | Execute command with bundle context |
| `bundle binstubs` | Generate binstub executables for gems |
| `bundle config` | Manage Bundler configuration settings |
| `bundle doctor` | Diagnose common Bundler issues |
| `bundle gem` | Create a new gem skeleton |
| `bundle cache` | Cache gems in vendor/cache directory |
| `bundle clean` | Remove unused cached gems |
| `bundle pristine` | Restore gems to pristine condition |
