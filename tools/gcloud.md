# Google Cloud (gcloud) Commands Cheatsheet

> gcloud is Google Cloud's official CLI. Control your GCP infrastructure from the terminal.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `gcloud init` | Initialize the gcloud CLI |
| `gcloud auth login` | Authenticate with Google Cloud |
| `gcloud config set` | Set a configuration property |
| `gcloud config list` | List current configuration properties |
| `gcloud auth application-default login` | Set up application default credentials |
| `gcloud components update` | Update installed gcloud components |
| `gcloud auth list` | List authenticated accounts |
| `gcloud compute instances create` | Create a Compute Engine instance |
| `gcloud compute instances list` | List all Compute Engine instances |
| `gcloud compute instances delete` | Delete a Compute Engine instance |
| `gcloud compute ssh` | SSH into a Compute Engine instance |
| `gcloud compute instances start` | Start a stopped instance |
| `gcloud compute instances stop` | Stop a running instance |
| `gcloud compute instances describe` | Show instance details |
| `gcloud compute firewall-rules create` | Create a firewall rule |
| `gsutil mb` | Create a Cloud Storage bucket |
| `gsutil ls` | List Cloud Storage buckets or objects |
| `gsutil cp` | Copy files to/from Cloud Storage |
| `gsutil rm` | Delete Cloud Storage objects |
| `gsutil rsync` | Sync local directory with Cloud Storage |
| `gsutil rb` | Remove a Cloud Storage bucket |
| `gcloud container clusters create` | Create a GKE cluster |
| `gcloud container clusters list` | List all GKE clusters |
| `gcloud container clusters get-credentials` | Get GKE cluster kubeconfig |
| `gcloud container clusters delete` | Delete a GKE cluster |
| `gcloud container clusters resize` | Resize a GKE cluster |
| `gcloud functions deploy` | Deploy a Cloud Function |
| `gcloud functions list` | List all Cloud Functions |
| `gcloud functions logs read` | Read Cloud Function logs |
| `gcloud run deploy` | Deploy a Cloud Run service |
| `gcloud run services list` | List Cloud Run services |
| `gcloud iam service-accounts create` | Create a service account |
| `gcloud iam service-accounts list` | List service accounts |
| `gcloud projects add-iam-policy-binding` | Add an IAM policy binding |
| `gcloud iam service-accounts keys create` | Create a service account key |
| `gcloud projects list` | List all GCP projects |
| `gcloud projects create` | Create a new GCP project |
| `gcloud services enable` | Enable a GCP API service |
| `gcloud services list` | List enabled API services |
| `gcloud logging read` | Read log entries |
