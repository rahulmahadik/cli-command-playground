# Vagrant Commands Cheatsheet

> Vagrant is a tool for building and managing virtual machine environments.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `vagrant init` | Initialize a new Vagrantfile |
| `vagrant up` | Create and start the virtual machine |
| `vagrant halt` | Gracefully shut down the VM |
| `vagrant destroy` | Stop and delete the VM |
| `vagrant suspend` | Suspend the VM |
| `vagrant resume` | Resume a suspended VM |
| `vagrant reload` | Restart the VM with updated config |
| `vagrant ssh` | SSH into the running VM |
| `vagrant ssh-config` | Show SSH configuration for the VM |
| `vagrant rdp` | Connect via RDP (Windows VMs) |
| `vagrant port` | Show forwarded port mappings |
| `vagrant status` | Show the status of the VM |
| `vagrant global-status` | Show status of all VMs system-wide |
| `vagrant validate` | Validate the Vagrantfile |
| `vagrant box add` | Add a box to the local repository |
| `vagrant box remove` | Remove a box |
| `vagrant box update` | Update a box to the latest version |
| `vagrant box list` | List downloaded boxes |
| `vagrant box outdated` | Check for outdated boxes |
| `vagrant provision` | Run provisioners on the VM |
| `vagrant snapshot save` | Save a named snapshot |
| `vagrant snapshot restore` | Restore a named snapshot |
| `vagrant snapshot list` | List available snapshots |
| `vagrant snapshot delete` | Delete a snapshot |
