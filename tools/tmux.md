# tmux Commands Cheatsheet

> tmux is a terminal multiplexer.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `tmux new-session` | Create a new tmux session |
| `tmux attach` | Attach to an existing session |
| `tmux detach` | Detach from the current session |
| `tmux list-sessions` | List all active sessions |
| `tmux kill-session` | Kill a session |
| `tmux rename-session` | Rename a session |
| `tmux new-window` | Create a new window |
| `tmux rename-window` | Rename the current window |
| `tmux kill-window` | Close a window |
| `tmux list-windows` | List windows in a session |
| `tmux select-window` | Switch to a specific window |
| `tmux split-window` | Split the current pane vertically |
| `tmux split-window -h` | Split the current pane horizontally |
| `tmux kill-pane` | Close the current pane |
| `tmux swap-pane` | Swap two panes |
| `tmux select-pane` | Switch to a specific pane |
| `Prefix + n / p` | Navigate to next/previous window |
| `Prefix + 0-9` | Switch to window by number |
| `Prefix + arrow keys` | Navigate between panes |
| `Prefix + s` | Show session chooser |
| `Prefix + w` | Show window chooser |
| `Prefix + [` | Enter copy/scroll mode |
| `tmux show-buffer` | Display the paste buffer |
| `tmux save-buffer` | Save buffer to a file |
| `tmux capture-pane` | Capture pane contents to a buffer |
| `Prefix + Space` | Cycle through pane layouts |
| `tmux resize-pane` | Resize a pane |
| `tmux select-layout` | Set a specific layout |
| `Prefix + z` | Toggle pane zoom |
| `tmux source-file` | Reload tmux configuration |
| `tmux set-option` | Set a tmux option |
| `tmux show-options` | Show current option values |
| `tmux bind-key` | Bind a key to a command |
| `tmux send-keys` | Send keystrokes to a pane |
| `tmux run-shell` | Run a shell command |
| `tmux display-message` | Display a message in the status line |
| `tmux if-shell` | Run command conditionally |
| `set -g status` | Toggle the status bar |
| `set -g status-style` | Set status bar style |
| `set -g status-left` | Configure left status bar |
| `set -g status-right` | Configure right status bar |
| `tmux pipe-pane` | Pipe pane output to a command |
| `tmux wait-for` | Wait for a signal |
| `tmux set-hook` | Set a hook for tmux events |
