# Micronaut Commands Cheatsheet

> Micronaut is a modern JVM framework for microservices.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `mn create-app` | Create a new Micronaut application |
| `mn create-function` | Create a serverless function project |
| `mn create-cli-app` | Create a command-line application |
| `mn create-grpc-app` | Create a gRPC-based application |
| `mn create-messaging-app` | Create a messaging-driven application |
| `mn create-controller` | Generate a new controller class |
| `mn create-bean` | Generate a new bean class |
| `mn create-client` | Generate a new HTTP client |
| `mn create-job` | Generate a scheduled job class |
| `mn create-websocket-server` | Generate a WebSocket server endpoint |
| `mn create-websocket-client` | Generate a WebSocket client |
| `mn feature-diff` | Show diff of adding a feature |
| `mn create-test` | Generate a test class |
| `mn build` | Build the Micronaut project |
| `mn test` | Run project tests |
