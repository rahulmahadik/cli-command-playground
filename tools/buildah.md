# Buildah Commands Cheatsheet

> Buildah is an OCI container image build tool.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `buildah from` | Create a new working container from an image |
| `buildah run` | Run a command in a working container |
| `buildah copy` | Copy files into the container |
| `buildah config` | Set configuration values for a container |
| `buildah commit` | Create an image from a working container |
| `buildah build` | Build an image using a Containerfile |
| `buildah containers` | List working containers |
| `buildah rm` | Remove a working container |
| `buildah images` | List locally stored images |
| `buildah push` | Push an image to a registry |
| `buildah pull` | Pull an image from a registry |
| `buildah login` | Log in to a container registry |
| `buildah inspect` | Inspect a container or image |
| `buildah tag` | Add a tag to an image |
| `buildah rmi` | Remove a locally stored image |
| `buildah add` | Add files or a URL to a container |
| `buildah mount` | Mount a working container filesystem |
| `buildah unshare` | Run a command in a user namespace |
