# Jenkins CLI Commands Cheatsheet

> Jenkins CLI is the command-line interface for the Jenkins automation server.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `jenkins-cli list-jobs` | List all jobs |
| `jenkins-cli create-job` | Create a new job from XML |
| `jenkins-cli delete-job` | Delete a job |
| `jenkins-cli copy-job` | Copy an existing job |
| `jenkins-cli enable-job` | Enable a disabled job |
| `jenkins-cli disable-job` | Disable a job |
| `jenkins-cli get-job` | Get the XML definition of a job |
| `jenkins-cli build` | Trigger a build |
| `jenkins-cli console` | View build console output |
| `jenkins-cli stop-build` | Stop a running build |
| `jenkins-cli set-build-result` | Set the result of a build |
| `jenkins-cli who-am-i` | Show the current user |
| `jenkins-cli groovy` | Execute a Groovy script |
| `jenkins-cli install-plugin` | Install a plugin |
| `jenkins-cli list-plugins` | List installed plugins |
| `jenkins-cli enable-plugin` | Enable a plugin |
| `jenkins-cli disable-plugin` | Disable a plugin |
| `jenkins-plugin-cli` | Manage plugins via the plugin CLI |
| `jenkins-cli safe-restart` | Restart Jenkins safely |
| `jenkins-cli safe-shutdown` | Shut down Jenkins safely |
| `jenkins-cli reload-configuration` | Reload configuration from disk |
| `jenkins-cli version` | Display the Jenkins version |
| `jenkins-cli connect-node` | Connect a build agent |
| `jenkins-cli disconnect-node` | Disconnect a build agent |
| `jenkins-cli online-node` | Bring a node online |
| `jenkins-cli offline-node` | Take a node offline |
| `jenkins-cli create-credentials-by-xml` | Create credentials from XML |
| `jenkins-cli list-credentials` | List stored credentials |
