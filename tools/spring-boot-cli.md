# Spring Boot Commands Cheatsheet

> Spring Boot is a Java framework for building web applications and microservices.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `spring --version` | Display installed Spring Boot version |
| `spring help` | Show help for Spring Boot CLI |
| `spring init` | Initialize a new Spring Boot project |
| `spring run` | Run a Spring Groovy script |
| `spring jar` | Package application into a JAR file |
| `spring shell` | Start an interactive Spring shell |
