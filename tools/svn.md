# SVN Commands Cheatsheet

> SVN is a centralized version control system.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `svnadmin create` | Create a new SVN repository |
| `svn import` | Import files into a repository |
| `svn checkout` | Check out a working copy |
| `svn status` | Show working copy status |
| `svn info` | Display information about a path |
| `svn cleanup` | Clean up the working copy |
| `svn export` | Export a clean directory tree |
| `svn add` | Schedule files for addition |
| `svn delete` | Schedule files for deletion |
| `svn copy` | Copy files or directories |
| `svn move` | Move or rename files |
| `svn revert` | Revert local changes |
| `svn commit` | Commit changes to the repository |
| `svn update` | Update the working copy |
| `svn merge` | Merge changes from another branch |
| `svn switch` | Switch working copy to a different URL |
| `svn resolve` | Resolve conflicts |
| `svn copy (branch)` | Create a branch via copy |
| `svn copy (tag)` | Create a tag via copy |
| `svn log` | View commit history |
| `svn blame` | Show per-line authorship |
| `svn cat` | Output file contents from the repository |
| `svn list` | List directory entries in the repository |
| `svn diff` | Show differences between revisions |
| `svn propset` | Set a property on a path |
| `svn propget` | Get a property value |
| `svn proplist` | List all properties |
| `svn propdel` | Delete a property |
| `svnadmin dump` | Dump repository data |
| `svnadmin load` | Load data into a repository |
| `svnadmin hotcopy` | Make a hot backup of a repository |
