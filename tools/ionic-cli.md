# Ionic Commands Cheatsheet

> Ionic is a cross-platform mobile app development framework.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `ionic start` | Create a new Ionic project |
| `ionic info` | Display Ionic environment information |
| `ionic generate` | Generate pages, components, and services |
| `ionic repair` | Repair project dependencies and plugins |
| `ionic serve` | Start local development server |
| `ionic serve --lab` | Serve with multi-platform preview |
| `ionic serve --external` | Serve accessible on local network |
| `ionic build` | Build the web application |
| `ionic build --prod` | Build optimized for production |
| `ionic build --configuration` | Build with specific configuration |
| `ionic capacitor add` | Add a native platform |
| `ionic capacitor add android` | Add Android native platform |
| `ionic capacitor add ios` | Add iOS native platform |
| `ionic capacitor run` | Run on native device or emulator |
| `ionic capacitor run android` | Run on Android device or emulator |
| `ionic capacitor run ios` | Run on iOS device or simulator |
| `ionic capacitor sync` | Sync web assets to native project |
| `ionic capacitor build` | Build native project for release |
| `ionic capacitor copy` | Copy web assets to native project |
| `ionic deploy` | Deploy app to Ionic Appflow |
| `ionic doctor` | Check project for common issues |
| `ionic login` | Log in to Ionic account |
