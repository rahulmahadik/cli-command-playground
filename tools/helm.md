# Helm Commands Cheatsheet

> Helm is the package manager for Kubernetes. Deploy complex Kubernetes applications with reusable charts.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `helm create` | Create a new Helm chart |
| `helm package` | Package a chart into an archive |
| `helm lint` | Validate a chart for errors |
| `helm dependency update` | Update chart dependencies |
| `helm show` | Show chart information |
| `helm install` | Install a Helm chart release |
| `helm upgrade` | Upgrade a deployed release |
| `helm uninstall` | Uninstall a deployed release |
| `helm rollback` | Roll back a release to a previous version |
| `helm list` | List all deployed releases |
| `helm repo add` | Add a chart repository |
| `helm repo update` | Update chart repository index |
| `helm repo list` | List added chart repositories |
| `helm repo remove` | Remove a chart repository |
| `helm search repo` | Search for charts in repositories |
| `helm status` | Show the status of a release |
| `helm history` | Show the revision history of a release |
| `helm get values` | Get the values of a deployed release |
| `helm get manifest` | Get the manifest of a release |
| `helm env` | Show Helm environment information |
| `helm template` | Render chart templates locally |
| `helm install --dry-run` | Simulate an install without deploying |
| `helm diff` | Show diff between release versions |
| `helm test` | Run tests for a release |
| `helm plugin install` | Install a Helm plugin |
| `helm plugin list` | List installed Helm plugins |
