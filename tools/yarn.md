# Yarn Commands Cheatsheet

> Yarn is a fast JavaScript package manager. Manage dependencies with deterministic installs.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `yarn install` | Install all project dependencies |
| `yarn add` | Add a package to dependencies |
| `yarn add --dev` | Add a package as a dev dependency |
| `yarn upgrade` | Upgrade a package to latest version |
| `yarn upgrade-interactive` | Interactively choose packages to upgrade |
| `yarn remove` | Remove a package from dependencies |
| `yarn info` | Show information about a package |
| `yarn list` | List all installed packages |
| `yarn outdated` | List outdated packages |
| `yarn why` | Show why a package is installed |
| `yarn run` | Run a script from package.json |
| `yarn start` | Run the start script |
| `yarn test` | Run the test script |
| `yarn build` | Run the build script |
| `yarn init` | Initialize a new package.json |
| `yarn create` | Create a new project from a starter |
| `yarn link` | Symlink a package for local development |
| `yarn unlink` | Remove a symlinked package |
| `yarn cache clean` | Clear the Yarn cache |
| `yarn cache list` | List all cached packages |
| `yarn cache dir` | Show the cache directory path |
| `yarn workspaces info` | Show workspace dependency info |
| `yarn workspace` | Run a command in a specific workspace |
| `yarn workspaces run` | Run a command in all workspaces |
| `yarn publish` | Publish a package to the registry |
| `yarn pack` | Create a tarball from a package |
| `yarn version` | Bump the package version |
| `yarn config set` | Set a Yarn configuration value |
| `yarn config get` | Get a Yarn configuration value |
| `yarn config list` | List all Yarn configuration |
| `yarn global add` | Install a package globally |
| `yarn global remove` | Remove a globally installed package |
| `yarn global list` | List globally installed packages |
| `yarn audit` | Run a security audit on dependencies |
