# Tauri Commands Cheatsheet

> Tauri is a Rust-based framework for building desktop applications.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `npm create tauri-app` | Create a new Tauri application |
| `cargo tauri dev` | Start Tauri app in development mode |
| `cargo tauri build` | Build Tauri app for production |
| `tauri init` | Initialize Tauri in existing project |
| `tauri info` | Display Tauri environment information |
