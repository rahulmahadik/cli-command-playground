# Serverless Framework Commands Cheatsheet

> Serverless Framework is an open-source toolkit for building serverless applications.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `serverless create` | Create a new serverless service |
| `serverless config credentials` | Configure provider credentials |
| `serverless deploy` | Deploy the entire service |
| `serverless deploy function` | Deploy a single function only |
| `serverless package` | Package service without deploying |
| `serverless rollback` | Rollback to a previous deployment |
| `serverless invoke` | Invoke a deployed function |
| `serverless invoke local` | Invoke a function locally |
| `serverless offline` | Run service offline for development |
| `serverless logs` | View function log output |
| `serverless info` | Display service information |
| `serverless print` | Print resolved serverless config |
| `serverless plugin` | Manage serverless plugins |
| `serverless remove` | Remove the deployed service |
