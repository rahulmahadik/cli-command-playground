# Apache Camel Commands Cheatsheet

> Apache Camel is an open-source integration framework.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `camel init` | Create a new Camel integration file |
| `camel init --directory` | Create a new project in a directory |
| `camel version` | Display the Camel CLI version |
| `camel run` | Run a Camel integration |
| `camel run --dev` | Run in developer mode with live reload |
| `camel run --port` | Run on a specific port |
| `camel run --profile` | Run with a specific profile |
| `camel get` | Get running Camel integrations |
| `camel stop` | Stop a running integration |
| `camel log` | Display integration logs |
| `camel cmd send` | Send a message to a running integration |
| `camel cmd route-stat` | Show route statistics |
| `camel catalog component` | Browse available components |
| `camel catalog language` | Browse available languages |
| `camel catalog dataformat` | Browse available data formats |
| `camel doc` | Show documentation for a component |
| `camel export` | Export integration to a standalone project |
| `camel export --runtime=spring-boot` | Export as a Spring Boot project |
| `camel export --runtime=quarkus` | Export as a Quarkus project |
| `camel export --gav` | Export with custom group/artifact/version |
| `camel cmd debug` | Debug a running integration |
| `camel cmd trace` | Trace message flow |
| `camel cmd top` | Show top-performing routes |
| `camel cmd source` | Display route source code |
| `camel dependency list` | List project dependencies |
| `camel dependency copy` | Copy dependencies locally |
| `camel dependency update` | Update project dependencies |
| `camel transform` | Transform data between formats |
| `camel cmd type-converter` | List type converters |
| `camel kubernetes run` | Run integration on Kubernetes |
| `camel kubernetes logs` | View logs on Kubernetes |
| `camel kubernetes delete` | Delete integration from Kubernetes |
| `camel config` | Manage CLI configuration |
| `camel plugin add` | Add a CLI plugin |
| `camel plugin list` | List installed CLI plugins |
