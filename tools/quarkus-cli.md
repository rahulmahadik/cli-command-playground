# Quarkus Commands Cheatsheet

> Quarkus is a Kubernetes-native Java framework.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `quarkus create` | Create a new Quarkus project |
| `quarkus create app` | Create a new Quarkus application |
| `quarkus dev` | Start app in development mode |
| `quarkus test` | Run continuous testing mode |
| `quarkus build` | Build the Quarkus application |
| `quarkus image build` | Build container image for application |
| `quarkus extension list` | List available Quarkus extensions |
| `quarkus extension add` | Add an extension to project |
| `quarkus extension remove` | Remove an extension from project |
| `quarkus deploy` | Deploy application to target platform |
