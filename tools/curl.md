# curl Commands Cheatsheet

> curl is a command-line data transfer tool. Transfer data over HTTP, FTP, SFTP, and more.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `curl` | Make a basic HTTP GET request |
| `curl -I` | Fetch HTTP headers only |
| `curl -L` | Follow HTTP redirects automatically |
| `curl -s` | Silent mode, suppress progress output |
| `curl -X GET` | Send an explicit GET request |
| `curl -X POST` | Send an HTTP POST request |
| `curl -X PUT` | Send an HTTP PUT request |
| `curl -X DELETE` | Send an HTTP DELETE request |
| `curl -X PATCH` | Send an HTTP PATCH request |
| `curl -H` | Set a custom HTTP header |
| `curl -u` | Authenticate with username and password |
| `curl --oauth2-bearer` | Authenticate with an OAuth2 token |
| `curl -A` | Set a custom User-Agent string |
| `curl -d` | Send POST data in the request body |
| `curl --json` | Send JSON data with proper headers |
| `curl -F` | Submit form data with file upload |
| `curl --data-urlencode` | Send URL-encoded POST data |
| `curl -o` | Save output to a named file |
| `curl -O` | Save output using the remote filename |
| `curl -C -` | Resume a previously interrupted download |
| `curl -T` | Upload a file to a server |
| `curl -k` | Allow insecure SSL connections |
| `curl --cacert` | Specify a CA certificate file |
| `curl --cert` | Use a client certificate file |
| `curl -v` | Verbose output with request details |
| `curl -w` | Display custom output after transfer |
| `curl --trace` | Full trace dump of all data |
| `curl --connect-timeout` | Set connection timeout in seconds |
| `curl --max-time` | Set maximum transfer time |
| `curl -x` | Use a proxy server |
| `curl --retry` | Retry on transient failures |
| `curl --limit-rate` | Limit the transfer speed |
| `curl -b` | Send cookies with the request |
| `curl -c` | Save cookies to a file |
