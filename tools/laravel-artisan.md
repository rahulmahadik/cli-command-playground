# Laravel Artisan Commands Cheatsheet

> Laravel Artisan is the built-in CLI for the Laravel PHP framework.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `php artisan serve` | Start local development server |
| `php artisan list` | List all available Artisan commands |
| `php artisan help` | Show help for a specific command |
| `php artisan key:generate` | Generate application encryption key |
| `php artisan about` | Display application environment overview |
| `php artisan make:controller` | Create a new controller class |
| `php artisan make:model` | Create a new Eloquent model |
| `php artisan make:migration` | Create a new database migration |
| `php artisan make:seeder` | Create a new database seeder |
| `php artisan make:factory` | Create a new model factory |
| `php artisan make:middleware` | Create a new middleware class |
| `php artisan make:request` | Create a new form request class |
| `php artisan make:resource` | Create a new API resource class |
| `php artisan make:command` | Create a new Artisan command |
| `php artisan make:event` | Create a new event class |
| `php artisan make:listener` | Create a new event listener |
| `php artisan make:job` | Create a new queued job class |
| `php artisan make:mail` | Create a new mailable class |
| `php artisan make:notification` | Create a new notification class |
| `php artisan make:policy` | Create a new authorization policy |
| `php artisan make:provider` | Create a new service provider |
| `php artisan migrate` | Run pending database migrations |
| `php artisan migrate:rollback` | Rollback the last batch of migrations |
| `php artisan migrate:reset` | Rollback all database migrations |
| `php artisan migrate:refresh` | Reset and re-run all migrations |
| `php artisan migrate:fresh` | Drop all tables and re-migrate |
| `php artisan migrate:status` | Show migration execution status |
| `php artisan db:seed` | Seed database with test records |
| `php artisan db:wipe` | Drop all tables, views, and types |
| `php artisan schema:dump` | Dump database schema to file |
| `php artisan route:list` | List all registered routes |
| `php artisan route:cache` | Cache route registrations for speed |
| `php artisan route:clear` | Clear the route cache file |
| `php artisan cache:clear` | Flush the application cache |
| `php artisan cache:forget` | Remove a specific cached item |
| `php artisan cache:table` | Create cache database table migration |
| `php artisan view:clear` | Clear compiled view template files |
| `php artisan view:cache` | Precompile all Blade view templates |
| `php artisan config:cache` | Cache configuration for faster loading |
| `php artisan config:clear` | Remove the configuration cache file |
| `php artisan config:publish` | Publish configuration files to app |
| `php artisan env:encrypt` | Encrypt the environment file |
| `php artisan env:decrypt` | Decrypt the environment file |
| `php artisan queue:work` | Start processing queued jobs |
| `php artisan queue:listen` | Listen for new queued jobs |
| `php artisan queue:restart` | Restart all queue worker daemons |
| `php artisan queue:retry` | Retry a failed queued job |
| `php artisan queue:failed` | List all failed queued jobs |
| `php artisan queue:flush` | Flush all failed queued jobs |
| `php artisan queue:table` | Create queue database table migration |
| `php artisan schedule:run` | Run scheduled commands manually |
| `php artisan schedule:list` | List all scheduled commands |
| `php artisan schedule:work` | Start the schedule worker daemon |
| `php artisan schedule:test` | Test a scheduled command |
| `php artisan test` | Run application test suite |
| `php artisan make:test` | Create a new test class |
| `php artisan test --filter` | Run tests matching a filter |
| `php artisan test --parallel` | Run tests in parallel processes |
| `php artisan tinker` | Start interactive REPL shell |
| `php artisan storage:link` | Create symbolic link for storage |
| `php artisan storage:unlink` | Remove the storage symbolic link |
| `php artisan vendor:publish` | Publish vendor package assets |
| `php artisan package:discover` | Rebuild package manifest cache |
| `php artisan down` | Put application in maintenance mode |
| `php artisan up` | Bring application out of maintenance |
| `php artisan optimize` | Cache routes, config, and views |
| `php artisan optimize:clear` | Clear all cached optimization files |
