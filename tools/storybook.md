# Storybook Commands Cheatsheet

> Storybook is a frontend workshop for developing UI components.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `npx storybook init` | Initialize Storybook in a project |
| `npm run storybook` | Start Storybook development server |
| `npm run build-storybook` | Build static Storybook for deployment |
| `storybook add` | Add a Storybook addon to project |
| `npx storybook upgrade` | Upgrade Storybook to latest version |
| `npx storybook doctor` | Diagnose Storybook configuration issues |
| `npx storybook automigrate` | Auto-migrate to newer Storybook config |
