# Rails CLI Commands Cheatsheet

> Rails CLI is the command-line interface for Ruby on Rails.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `rails new` | Create a new Rails application |
| `rails new --api` | Create an API-only application |
| `rails new --database` | Create with a specific database |
| `rails new --css` | Create with a CSS framework |
| `rails generate model` | Generate a model |
| `rails generate controller` | Generate a controller |
| `rails generate scaffold` | Generate a full CRUD scaffold |
| `rails generate migration` | Generate a database migration |
| `rails generate mailer` | Generate a mailer |
| `rails destroy` | Undo a generate command |
| `rails server` | Start the development server |
| `rails console` | Open the Rails interactive console |
| `rails dbconsole` | Open the database console |
| `rails db:create` | Create the database |
| `rails db:migrate` | Run pending migrations |
| `rails db:seed` | Load seed data |
| `rails db:rollback` | Roll back the last migration |
| `rails db:reset` | Drop, create, and migrate the database |
| `rails db:schema:load` | Load the schema file |
| `rails test` | Run the test suite |
| `rails test:models` | Run model tests |
| `rails test:system` | Run system tests |
| `rails assets:precompile` | Precompile assets for production |
| `rails routes` | List all routes |
| `rails middleware` | List middleware stack |
| `rails credentials:edit` | Edit encrypted credentials |
| `rails runner` | Run Ruby code in the Rails context |
| `rails notes` | List code annotations |
| `rails stats` | Show code statistics |
