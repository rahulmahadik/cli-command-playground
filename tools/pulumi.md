# Pulumi Commands Cheatsheet

> Pulumi is an infrastructure-as-code platform.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `pulumi new` | Create a new Pulumi project |
| `pulumi stack init` | Create a new deployment stack |
| `pulumi stack ls` | List all available stacks |
| `pulumi stack select` | Switch to a different stack |
| `pulumi stack output` | Show stack output values |
| `pulumi stack rm` | Remove a deployment stack |
| `pulumi config` | View stack configuration values |
| `pulumi config set` | Set a configuration value |
| `pulumi config set --secret` | Set an encrypted secret value |
| `pulumi config get` | Get a configuration value |
| `pulumi config rm` | Remove a configuration value |
| `pulumi preview` | Preview infrastructure changes |
| `pulumi up` | Deploy infrastructure changes |
| `pulumi destroy` | Destroy all managed infrastructure |
| `pulumi cancel` | Cancel a currently running update |
| `pulumi refresh` | Refresh state against actual resources |
| `pulumi stack export` | Export stack state to file |
| `pulumi stack import` | Import stack state from file |
| `pulumi login` | Log in to Pulumi backend |
| `pulumi import` | Import existing resource into state |
