# Cypress Commands Cheatsheet

> Cypress is an end-to-end testing framework for web applications.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `npm install cypress` | Install Cypress as a dependency |
| `npx cypress install` | Install Cypress binary separately |
| `npx cypress run` | Run all tests in headless mode |
| `npx cypress run --spec` | Run specific test spec file |
| `npx cypress run --browser` | Run tests in a specific browser |
| `npx cypress run --headed` | Run tests in headed browser mode |
| `npx cypress run --record` | Record test run to Cypress Cloud |
| `npx cypress run --parallel` | Run tests in parallel across machines |
| `npx cypress open` | Open Cypress interactive test runner |
| `npx cypress open --e2e` | Open runner for end-to-end tests |
| `npx cypress open --component` | Open runner for component tests |
| `npx cypress open --browser` | Open runner with specific browser |
| `npx cypress run --config` | Run with custom configuration values |
| `npx cypress run --env` | Run with custom environment variables |
| `npx cypress verify` | Verify Cypress installation is correct |
| `npx cypress info` | Display Cypress system information |
| `npx cypress version` | Show installed Cypress version |
| `npx cypress cache clear` | Clear Cypress binary cache |
| `npx cypress cache list` | List cached Cypress binary versions |
| `npx cypress run --reporter` | Run tests with custom reporter |
| `npx cypress run --config-file` | Run with alternate config file |
