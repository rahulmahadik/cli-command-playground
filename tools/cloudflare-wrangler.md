# Cloudflare Wrangler Commands Cheatsheet

> Cloudflare Wrangler is the CLI for Cloudflare Workers and Pages.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `wrangler login` | Authenticate with Cloudflare |
| `wrangler logout` | Log out of Cloudflare |
| `wrangler whoami` | Display the current user |
| `wrangler init` | Initialize a new Worker project |
| `wrangler generate` | Generate a project from a template |
| `wrangler dev` | Start a local development server |
| `wrangler deploy` | Deploy the Worker to Cloudflare |
| `wrangler publish` | Publish the Worker (legacy) |
| `wrangler tail` | Stream real-time logs from a Worker |
| `wrangler pages deploy` | Deploy a Pages project |
| `wrangler pages project create` | Create a new Pages project |
| `wrangler secret put` | Set a secret for a Worker |
| `wrangler secret list` | List secrets for a Worker |
| `wrangler secret delete` | Delete a secret |
| `wrangler kv namespace create` | Create a KV namespace |
| `wrangler kv namespace list` | List KV namespaces |
| `wrangler kv key put` | Write a key-value pair |
| `wrangler kv key get` | Read a value by key |
| `wrangler kv key list` | List keys in a namespace |
| `wrangler kv key delete` | Delete a key |
| `wrangler d1 create` | Create a D1 database |
| `wrangler d1 execute` | Execute SQL on a D1 database |
| `wrangler d1 migrations apply` | Apply D1 migrations |
| `wrangler d1 migrations create` | Create a new D1 migration |
| `wrangler d1 list` | List D1 databases |
| `wrangler r2 bucket create` | Create an R2 bucket |
| `wrangler r2 bucket list` | List R2 buckets |
| `wrangler r2 object put` | Upload an object to R2 |
| `wrangler r2 object get` | Download an object from R2 |
