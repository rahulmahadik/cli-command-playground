# Docker Commands Cheatsheet

> Docker is a platform for containerizing applications. Build, ship, and run apps in lightweight containers.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `docker run` | Create and start a new container |
| `docker create` | Create a new container without starting |
| `docker start` | Start one or more stopped containers |
| `docker stop` | Stop one or more running containers |
| `docker restart` | Restart one or more containers |
| `docker rm` | Remove one or more containers |
| `docker kill` | Kill a running container immediately |
| `docker pause` | Pause all processes in a container |
| `docker unpause` | Unpause a paused container |
| `docker wait` | Block until a container stops |
| `docker rename` | Rename a container |
| `docker update` | Update container resource configuration |
| `docker ps` | List running containers |
| `docker logs` | Fetch the logs of a container |
| `docker inspect` | Return low-level container or image info |
| `docker top` | Display running processes of a container |
| `docker stats` | Display live resource usage statistics |
| `docker diff` | Inspect filesystem changes in a container |
| `docker port` | List port mappings for a container |
| `docker attach` | Attach to a running container |
| `docker build` | Build an image from a Dockerfile |
| `docker pull` | Pull an image from a registry |
| `docker push` | Push an image to a registry |
| `docker images` | List locally available images |
| `docker rmi` | Remove one or more images |
| `docker tag` | Tag an image for a repository |
| `docker history` | Show the history of an image |
| `docker save` | Save images to a tar archive |
| `docker load` | Load images from a tar archive |
| `docker import` | Import contents from a tarball |
| `docker image prune` | Remove unused images |
| `docker network create` | Create a new network |
| `docker network ls` | List all networks |
| `docker network rm` | Remove one or more networks |
| `docker network connect` | Connect a container to a network |
| `docker network disconnect` | Disconnect a container from a network |
| `docker network inspect` | Display detailed network information |
| `docker network prune` | Remove all unused networks |
| `docker volume create` | Create a new volume |
| `docker volume ls` | List all volumes |
| `docker volume rm` | Remove one or more volumes |
| `docker volume inspect` | Display detailed volume information |
| `docker volume prune` | Remove all unused volumes |
| `docker compose up` | Create and start compose services |
| `docker compose down` | Stop and remove compose services |
| `docker compose ps` | List compose service containers |
| `docker compose logs` | View output from compose services |
| `docker compose build` | Build compose service images |
| `docker compose exec` | Execute a command in a running service |
| `docker compose run` | Run a one-off command in a service |
| `docker compose stop` | Stop compose services |
| `docker compose start` | Start existing compose services |
| `docker compose restart` | Restart compose services |
| `docker compose pull` | Pull compose service images |
| `docker compose config` | Validate and view compose configuration |
| `docker info` | Display system-wide Docker information |
| `docker version` | Show Docker version information |
| `docker system df` | Show Docker disk usage |
| `docker system prune` | Remove all unused data |
| `docker system events` | Get real-time Docker events |
| `docker login` | Log in to a Docker registry |
| `docker logout` | Log out from a Docker registry |
| `docker search` | Search Docker Hub for images |
| `docker exec` | Run a command in a running container |
| `docker cp` | Copy files between container and host |
| `docker export` | Export container filesystem as tar |
| `docker commit` | Create image from container changes |
| `docker context` | Manage Docker contexts |
| `docker buildx` | Extended build capabilities with BuildKit |
| `docker manifest` | Manage Docker image manifests |
| `docker container prune` | Remove all stopped containers |
