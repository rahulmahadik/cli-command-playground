# GitLab CLI Commands Cheatsheet

> GitLab CLI (glab) is the official command-line tool for GitLab.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `glab auth login` | Authenticate with GitLab instance |
| `glab auth status` | Check authentication status |
| `glab auth logout` | Log out from GitLab |
| `glab config set` | Set a configuration value |
| `glab config get` | Get a configuration value |
| `glab config list` | List all configuration settings |
| `glab repo clone` | Clone a GitLab repository |
| `glab repo create` | Create a new GitLab repository |
| `glab repo view` | View repository details in browser |
| `glab repo fork` | Fork a GitLab repository |
| `glab repo list` | List your GitLab repositories |
| `glab repo archive` | Archive a GitLab repository |
| `glab repo delete` | Delete a GitLab repository |
| `glab mr create` | Create a new merge request |
| `glab mr list` | List open merge requests |
| `glab mr view` | View merge request details |
| `glab mr checkout` | Checkout merge request branch locally |
| `glab mr merge` | Merge a merge request |
| `glab mr approve` | Approve a merge request |
| `glab mr close` | Close a merge request |
| `glab mr reopen` | Reopen a closed merge request |
| `glab mr update` | Update merge request properties |
| `glab mr diff` | View merge request diff output |
| `glab mr note` | Add a comment to merge request |
| `glab issue create` | Create a new issue |
| `glab issue list` | List project issues |
| `glab issue view` | View issue details |
| `glab issue close` | Close an issue |
| `glab issue reopen` | Reopen a closed issue |
| `glab issue update` | Update issue properties |
| `glab issue note` | Add a comment to an issue |
| `glab issue subscribe` | Subscribe to issue notifications |
| `glab ci status` | Show CI pipeline status |
| `glab ci list` | List CI pipeline runs |
| `glab ci view` | View CI pipeline details |
| `glab ci run` | Trigger a CI pipeline run |
| `glab ci trace` | View CI job log output |
| `glab ci retry` | Retry a failed CI job |
| `glab ci cancel` | Cancel a running CI pipeline |
| `glab ci lint` | Lint the CI configuration file |
| `glab release create` | Create a new project release |
| `glab release list` | List project releases |
| `glab release view` | View release details |
| `glab release delete` | Delete a project release |
| `glab release download` | Download release assets |
| `glab snippet create` | Create a new code snippet |
| `glab snippet list` | List project code snippets |
| `glab snippet view` | View snippet content |
| `glab snippet delete` | Delete a code snippet |
| `glab api` | Make authenticated API requests |
| `glab variable list` | List CI/CD variables |
| `glab variable set` | Set a CI/CD variable |
| `glab variable get` | Get a CI/CD variable value |
| `glab variable delete` | Delete a CI/CD variable |
| `glab ssh-key list` | List SSH keys on account |
| `glab ssh-key add` | Add an SSH key to account |
