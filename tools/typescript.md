# TypeScript Commands Cheatsheet

> TypeScript is a typed superset of JavaScript. Add static types to your JS code to catch bugs early.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `tsc` | Compile TypeScript files to JavaScript |
| `tsc --target` | Set the JavaScript version target |
| `tsc --outDir` | Set the output directory for compiled files |
| `tsc --init` | Create a tsconfig.json configuration file |
| `tsc --project` | Compile using a specific tsconfig file |
| `tsc --strict` | Enable all strict type-checking options |
| `tsc --noEmit` | Type-check without emitting output files |
| `tsc --watch` | Watch for file changes and recompile |
| `tsc --build` | Build project references incrementally |
| `tsc --declaration` | Generate .d.ts type declaration files |
| `tsc --version` | Display the TypeScript compiler version |
| `tsc --help` | Show TypeScript compiler help information |
