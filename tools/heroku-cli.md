# Heroku Commands Cheatsheet

> Heroku is a cloud platform as a service.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `heroku login` | Authenticate with Heroku account |
| `heroku logout` | Log out from Heroku CLI |
| `heroku auth:whoami` | Display current authenticated user |
| `heroku auth:token` | Display current authentication token |
| `heroku create` | Create a new Heroku application |
| `heroku apps` | List all your Heroku applications |
| `heroku apps:info` | Show detailed application information |
| `heroku apps:rename` | Rename a Heroku application |
| `heroku apps:destroy` | Permanently delete an application |
| `heroku maintenance` | Toggle application maintenance mode |
| `heroku domains` | Manage custom domains for app |
| `git push heroku` | Deploy code via Git push |
| `heroku releases` | List application release history |
| `heroku releases:rollback` | Rollback to a previous release |
| `heroku container:push` | Push Docker image to Heroku |
| `heroku container:release` | Release pushed container image |
| `heroku ps` | List running application dynos |
| `heroku ps:scale` | Scale dyno formation up or down |
| `heroku ps:restart` | Restart application dynos |
| `heroku ps:stop` | Stop a specific running dyno |
| `heroku config` | Display app config environment variables |
| `heroku config:set` | Set config environment variables |
| `heroku config:unset` | Remove config environment variables |
| `heroku addons` | List installed application addons |
| `heroku addons:create` | Install a new addon |
| `heroku addons:destroy` | Remove an installed addon |
| `heroku logs` | View application log stream |
| `heroku run` | Run a one-off command on dyno |
| `heroku pg:psql` | Open PostgreSQL interactive shell |
| `heroku pg:info` | Display database connection info |
| `heroku pg:backups` | Manage database backup schedules |
