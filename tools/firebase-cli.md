# Firebase Commands Cheatsheet

> Firebase is Google's platform for building mobile and web apps.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `firebase login` | Authenticate with your Google account |
| `firebase login:ci` | Generate CI authentication token |
| `firebase logout` | Sign out of Firebase CLI |
| `firebase login:list` | List all authenticated accounts |
| `firebase init` | Initialize Firebase in a project |
| `firebase init hosting` | Set up Firebase Hosting configuration |
| `firebase init functions` | Set up Cloud Functions for Firebase |
| `firebase init firestore` | Set up Firestore database rules |
| `firebase projects:list` | List all Firebase projects |
| `firebase use` | Set active Firebase project |
| `firebase deploy` | Deploy all resources to Firebase |
| `firebase deploy --only hosting` | Deploy only hosting assets |
| `firebase deploy --only functions` | Deploy only Cloud Functions |
| `firebase deploy --only firestore` | Deploy only Firestore rules |
| `firebase serve` | Start local development server |
| `firebase serve --only hosting` | Serve only hosting locally |
| `firebase serve --only functions` | Serve only functions locally |
| `firebase open` | Open Firebase project in browser |
| `firebase emulators:start` | Start all Firebase emulators |
| `firebase emulators:exec` | Run script with emulators running |
| `firebase emulators:export` | Export emulator data to directory |
| `firebase emulators:import` | Import data into emulators |
| `firebase hosting:channel:create` | Create a preview channel |
| `firebase hosting:channel:deploy` | Deploy to a preview channel |
| `firebase hosting:channel:list` | List all preview channels |
| `firebase hosting:disable` | Disable Firebase Hosting |
| `firebase functions:log` | View Cloud Functions log output |
| `firebase functions:delete` | Delete a deployed Cloud Function |
| `firebase functions:config:set` | Set environment config variables |
| `firebase functions:config:get` | Get environment config variables |
| `firebase firestore:delete` | Delete Firestore documents or collections |
| `firebase firestore:indexes` | List Firestore composite indexes |
| `firebase database:get` | Read data from Realtime Database |
| `firebase database:set` | Write data to Realtime Database |
| `firebase database:push` | Push data to Realtime Database |
| `firebase database:remove` | Remove data from Realtime Database |
| `firebase storage:list` | List files in Cloud Storage |
| `firebase storage:delete` | Delete files from Cloud Storage |
