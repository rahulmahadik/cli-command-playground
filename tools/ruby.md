# Ruby Commands Cheatsheet

> Ruby is a dynamic, object-oriented scripting language.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `ruby` | Run a Ruby script file |
| `ruby -e` | Execute Ruby code from the command line |
| `ruby -c` | Check syntax without executing |
| `ruby -w` | Enable warnings |
| `ruby -v` | Display Ruby version |
| `irb` | Start the interactive Ruby shell |
| `irb --simple-prompt` | Start IRB with a simplified prompt |
| `irb -r` | Require a library before starting IRB |
| `gem install` | Install a Ruby gem |
| `gem uninstall` | Uninstall a Ruby gem |
| `gem list` | List installed gems |
| `gem search` | Search for available gems |
| `gem update` | Update installed gems |
| `gem environment` | Show gem environment info |
| `gem build` | Build a gem from a gemspec |
| `bundle init` | Create a new Gemfile |
| `bundle install` | Install gems from the Gemfile |
| `bundle update` | Update gems to latest versions |
| `bundle exec` | Execute a command in the bundle context |
| `bundle add` | Add a gem to the Gemfile |
| `bundle outdated` | List outdated gems |
| `rake` | Run the default Rake task |
| `rake -T` | List available Rake tasks |
| `rake -P` | Show task prerequisites |
| `rake --trace` | Run with full backtrace on errors |
| `rbenv install` | Install a Ruby version |
| `rbenv versions` | List installed Ruby versions |
| `rbenv global` | Set the global Ruby version |
| `rbenv local` | Set the local Ruby version |
| `rbenv rehash` | Rebuild the shim binaries |
| `ruby -ne` | Process input line by line |
| `ruby -pe` | Process and print each input line |
| `ruby -i` | Edit files in place |
| `ruby -an` | Auto-split mode with line processing |
| `ruby -rdebug` | Start with the debugger |
| `ruby -rbenchmark` | Load the benchmark library |
| `rspec` | Run RSpec test suite |
| `rspec --format` | Run tests with a specific output format |
| `ruby -Itest` | Add test directory to load path |
| `rake test` | Run tests via Rake |
| `ri` | Look up Ruby documentation |
| `rdoc` | Generate Ruby documentation |
| `ruby --help` | Display Ruby help information |
