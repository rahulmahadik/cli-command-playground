# Deno Commands Cheatsheet

> Deno is a modern, secure runtime for JavaScript and TypeScript.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `deno run` | Run a JavaScript/TypeScript file |
| `deno run --allow-net` | Run with network access permission |
| `deno task` | Run a task defined in deno.json |
| `deno serve` | Start an HTTP server |
| `deno install` | Install dependencies |
| `deno add` | Add a dependency to the project |
| `deno remove` | Remove a dependency |
| `deno cache` | Cache remote dependencies |
| `deno test` | Run tests |
| `deno lint` | Lint source files |
| `deno fmt` | Format source files |
| `deno check` | Type-check without running |
| `deno init` | Initialize a new Deno project |
| `deno.json (Configuration)` | Project configuration file |
| `deno compile` | Compile to a standalone executable |
| `deno bench` | Run benchmarks |
| `deno info` | Show dependency information |
| `deno doc` | Generate documentation |
| `deno repl` | Start the interactive REPL |
| `deno upgrade` | Upgrade Deno to the latest version |
| `deno eval` | Evaluate code from the command line |
| `deno run --allow-read` | Run with file read permission |
| `deno run --allow-write` | Run with file write permission |
| `deno run --allow-env` | Run with environment variable access |
| `deno run -A` | Run with all permissions |
