# Argo CD Commands Cheatsheet

> Argo CD is a GitOps continuous delivery tool for Kubernetes.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `argocd login` | Authenticate with Argo CD server |
| `argocd logout` | Log out from Argo CD server |
| `argocd context` | Switch between Argo CD server contexts |
| `argocd app create` | Create a new Argo CD application |
| `argocd app list` | List all managed applications |
| `argocd app sync` | Sync application to target state |
| `argocd app get` | Get application details and status |
| `argocd app delete` | Delete an Argo CD application |
| `argocd app diff` | Show diff between live and target |
| `argocd app history` | View application deployment history |
| `argocd app rollback` | Rollback to a previous app version |
| `argocd app set` | Update application parameters |
| `argocd app wait` | Wait for app to reach synced state |
| `argocd app logs` | View application pod logs |
| `argocd repo add` | Add a Git repository to Argo CD |
| `argocd repo list` | List configured Git repositories |
| `argocd repo rm` | Remove a Git repository |
| `argocd cluster add` | Add a Kubernetes cluster |
| `argocd cluster list` | List registered Kubernetes clusters |
| `argocd cluster rm` | Remove a Kubernetes cluster |
| `argocd proj create` | Create a new Argo CD project |
| `argocd proj list` | List all Argo CD projects |
| `argocd proj get` | Get project details and settings |
| `argocd proj delete` | Delete an Argo CD project |
| `argocd account list` | List Argo CD user accounts |
| `argocd account generate-token` | Generate API authentication token |
| `argocd account update-password` | Update user account password |
| `argocd version` | Display Argo CD client and server version |
