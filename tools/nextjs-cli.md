# Next.js Commands Cheatsheet

> Next.js is a React framework for production web applications. Build full-stack apps with file-based routing.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `npx create-next-app` | Scaffold a new Next.js application |
| `next dev` | Start the development server |
| `next build` | Create an optimized production build |
| `next start` | Start the production server |
| `next export` | Export the app as static HTML |
| `next lint` | Run ESLint on the project |
| `next info` | Print system and project information |
