# Bun Commands Cheatsheet

> Bun is an all-in-one JavaScript runtime, bundler, test runner, and package manager.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `bun run` | Run a JavaScript/TypeScript file or script |
| `bun --watch` | Run with automatic file watching |
| `bun --hot` | Run with hot module reloading |
| `bun --env-file` | Load environment variables from a file |
| `bun install` | Install all dependencies |
| `bun add` | Add a dependency |
| `bun remove` | Remove a dependency |
| `bun update` | Update dependencies |
| `bun pm` | Package manager utilities |
| `bun link` | Link a local package |
| `bun init` | Initialize a new project |
| `bun create` | Create a project from a template |
| `bun test` | Run tests |
| `bun test --watch` | Run tests in watch mode |
| `bun test --coverage` | Run tests with code coverage |
| `bun build` | Bundle JavaScript/TypeScript files |
| `bun build --minify` | Bundle with minification |
| `bun build --target` | Bundle for a specific target |
| `bun build --compile` | Compile to a standalone executable |
| `bunx` | Execute a package binary (like npx) |
| `bun upgrade` | Upgrade Bun to the latest version |
| `bun --version` | Display the Bun version |
| `bun completions` | Generate shell completions |
