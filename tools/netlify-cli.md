# Netlify Commands Cheatsheet

> Netlify is a platform for building and deploying web projects.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `netlify login` | Authenticate with Netlify account |
| `netlify status` | Show current Netlify connection status |
| `netlify logout` | Log out from Netlify CLI |
| `netlify init` | Initialize a new Netlify site |
| `netlify link` | Link directory to existing Netlify site |
| `netlify unlink` | Unlink directory from Netlify site |
| `netlify dev` | Start local development server |
| `netlify build` | Build the project locally |
| `netlify deploy` | Deploy to a draft URL |
| `netlify deploy --prod` | Deploy directly to production |
| `netlify deploy --dir` | Deploy a specific output directory |
| `netlify sites:list` | List all Netlify sites |
| `netlify sites:create` | Create a new Netlify site |
| `netlify sites:delete` | Delete a Netlify site |
| `netlify open` | Open site in the browser |
| `netlify env:list` | List environment variables |
| `netlify env:set` | Set an environment variable |
| `netlify env:get` | Get an environment variable value |
| `netlify env:unset` | Remove an environment variable |
| `netlify functions:list` | List deployed serverless functions |
| `netlify functions:create` | Create a new serverless function |
| `netlify functions:serve` | Serve functions locally |
| `netlify functions:invoke` | Invoke a function locally |
| `netlify addons:list` | List installed site addons |
| `netlify addons:create` | Install a new site addon |
| `netlify addons:delete` | Remove a site addon |
| `netlify forms:list` | List site form submissions |
| `netlify submissions:list` | List form submission entries |
| `netlify logs:deploy` | View deploy log output |
| `netlify logs:function` | View function execution logs |
| `netlify plugins:list` | List installed CLI plugins |
| `netlify plugins:install` | Install a CLI plugin |
| `netlify watch` | Watch for deploy completion |
| `netlify api` | Make raw Netlify API calls |
