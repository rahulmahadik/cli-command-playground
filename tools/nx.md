# Nx Commands Cheatsheet

> Nx is a powerful build system for monorepos.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `npx create-nx-workspace` | Create a new Nx workspace |
| `nx generate` | Generate code using Nx generators |
| `nx build` | Build a project in workspace |
| `nx serve` | Serve a project for development |
| `nx test` | Run tests for a project |
| `nx affected` | Run tasks on affected projects |
| `nx graph` | Visualize project dependency graph |
| `nx run-many` | Run tasks across multiple projects |
| `nx migrate` | Migrate workspace to newer versions |
| `nx list` | List installed and available plugins |
| `nx reset` | Reset Nx workspace cache |
