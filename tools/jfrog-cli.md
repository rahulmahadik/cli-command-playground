# JFrog CLI Commands Cheatsheet

> JFrog CLI is the universal command-line tool for the JFrog Platform.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `jf config add` | Add a JFrog Platform server configuration |
| `jf config show` | Display server configuration details |
| `jf config use` | Set the active server configuration |
| `jf rt upload` | Upload artifacts to Artifactory |
| `jf rt download` | Download artifacts from Artifactory |
| `jf rt search` | Search for artifacts in Artifactory |
| `jf rt copy` | Copy artifacts within Artifactory |
| `jf rt delete` | Delete artifacts from Artifactory |
| `jf rt move` | Move artifacts within Artifactory |
| `jf rt build-publish` | Publish build info to Artifactory |
| `jf rt build-add-dependencies` | Add dependencies to a build |
| `jf rt build-scan` | Scan a build with Xray |
| `jf rt build-promote` | Promote a build to another repository |
| `jf docker push` | Push a Docker image via Artifactory |
| `jf docker pull` | Pull a Docker image via Artifactory |
| `jf docker scan` | Scan a Docker image for vulnerabilities |
| `jf audit` | Audit project dependencies for issues |
| `jf scan` | Scan files or builds with Xray |
| `jf rt set-props` | Set properties on artifacts |
| `jf rt curl` | Execute a raw REST API call |
| `jf rt ping` | Ping the Artifactory server |
| `jf rt upload --spec` | Upload using a file spec |
