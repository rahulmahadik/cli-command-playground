# Bitbucket CLI Commands Cheatsheet

> Bitbucket CLI is Atlassian's command-line tool for Bitbucket.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `bitbucket auth login` | Authenticate with Bitbucket |
| `bitbucket auth status` | Check authentication status |
| `bitbucket auth logout` | Log out from Bitbucket |
| `bitbucket repo clone` | Clone a Bitbucket repository |
| `bitbucket repo create` | Create a new repository |
| `bitbucket repo view` | View repository details |
| `bitbucket repo list` | List repositories |
| `bitbucket repo fork` | Fork a repository |
| `bitbucket repo delete` | Delete a repository |
| `bitbucket pr create` | Create a pull request |
| `bitbucket pr list` | List pull requests |
| `bitbucket pr view` | View pull request details |
| `bitbucket pr checkout` | Check out a pull request locally |
| `bitbucket pr merge` | Merge a pull request |
| `bitbucket pr approve` | Approve a pull request |
| `bitbucket pr decline` | Decline a pull request |
| `bitbucket pr diff` | View pull request diff |
| `bitbucket pipelines list` | List pipeline runs |
| `bitbucket pipelines view` | View pipeline run details |
| `bitbucket pipelines run` | Trigger a pipeline run |
| `bitbucket pipelines stop` | Stop a running pipeline |
| `bitbucket pipelines variables` | Manage pipeline variables |
| `bitbucket branch list` | List branches |
| `bitbucket branch create` | Create a new branch |
| `bitbucket branch delete` | Delete a branch |
| `bitbucket deployments list` | List deployments |
