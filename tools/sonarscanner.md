# SonarScanner Commands Cheatsheet

> SonarScanner is the CLI for SonarQube code quality analysis.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `sonar-scanner` | Run code analysis with default settings |
| `sonar-scanner --help` | Display SonarScanner help information |
| `sonar-scanner with properties` | Run analysis with project properties file |
| `sonar-scanner -Dsonar.host.url` | Specify SonarQube server URL |
| `sonar-scanner -Dsonar.login` | Authenticate with login credentials |
| `sonar-scanner -Dsonar.token` | Authenticate with analysis token |
| `sonar-scanner -Dsonar.branch.name` | Analyze a specific branch |
| `sonar-scanner -Dsonar.exclusions` | Exclude files from analysis |
| `sonar-scanner -Dsonar.qualitygate.wait=true` | Wait for quality gate result |
| `sonar-scanner -X` | Run analysis with debug logging |
