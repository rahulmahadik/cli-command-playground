# Python Commands Cheatsheet

> Python is a high-level, general-purpose programming language. Write scripts, build web backends, analyze data.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `python script.py` | Execute a Python script file |
| `python -c` | Execute Python code from the command line |
| `python -u` | Run with unbuffered stdout and stderr |
| `python -` | Read script from standard input |
| `python` | Start interactive Python interpreter |
| `python -i` | Inspect interactively after running script |
| `python -q` | Start interpreter without version banner |
| `python -m` | Run a module as a script |
| `python -m http.server` | Start a simple HTTP server |
| `python -m json.tool` | Format and validate JSON data |
| `python -m pip` | Run pip package manager as module |
| `python -m pytest` | Run pytest testing framework |
| `python -m zipfile` | Create and extract zip archives |
| `python -m compileall` | Compile all Python source files |
| `python -m ensurepip` | Bootstrap the pip installer |
| `python -m pdb` | Run the Python debugger |
| `python -m cProfile` | Profile Python script performance |
| `python -m timeit` | Measure execution time of statements |
| `python -m trace` | Trace Python statement execution |
| `python -E` | Ignore Python environment variables |
| `python -s` | Skip user site-packages directory |
| `python -S` | Skip importing the site module |
| `python -O` | Enable basic optimizations |
| `python -OO` | Remove docstrings in addition to -O |
| `python -B` | Prevent writing .pyc bytecode files |
| `python --version` | Display the Python version |
| `python -V` | Display version (short form) |
| `python --help` | Show Python command-line help |
| `python -m venv` | Create a virtual environment |
| `source venv/bin/activate` | Activate a virtual environment |
| `deactivate` | Deactivate the current virtual environment |
| `python -W` | Control warning message behavior |
