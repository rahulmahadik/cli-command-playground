# Fly.io Commands Cheatsheet

> Fly.io CLI is the command-line tool for the Fly.io cloud platform.

## Quick Reference

> **Preview** — This is a curated selection of key commands. Explore the full command list with examples, practice exercises, and more on [Command Playground](https://technoscripts.com/command-playground/?utm_source=github&utm_medium=repository&utm_campaign=explore-all).

| Command | Description |
|---------|-------------|
| `fly auth login` | Authenticate with Fly.io |
| `fly auth logout` | Log out of Fly.io |
| `fly auth token` | Display the current auth token |
| `fly launch` | Create and configure a new Fly app |
| `fly apps list` | List all Fly applications |
| `fly status` | Show the status of an app |
| `fly apps destroy` | Delete an application |
| `fly open` | Open the app in a browser |
| `fly deploy` | Deploy the application |
| `fly releases` | List application releases |
| `fly logs` | View application logs |
| `fly ssh console` | Open an SSH session to a running machine |
| `fly machine list` | List machines for an app |
| `fly proxy` | Proxy a local port to a remote app |
| `fly scale count` | Scale the number of machines |
| `fly scale memory` | Set the memory size for machines |
| `fly scale vm` | Set the VM size for machines |
| `fly regions add` | Add deployment regions |
| `fly regions list` | List configured regions |
| `fly secrets set` | Set an application secret |
| `fly secrets list` | List application secrets |
| `fly secrets unset` | Remove an application secret |
| `fly ips list` | List allocated IP addresses |
| `fly volumes create` | Create a persistent volume |
| `fly volumes list` | List persistent volumes |
| `fly postgres create` | Create a managed Postgres cluster |
| `fly postgres attach` | Attach a Postgres database to an app |
