## How It Works

1. **Pick a tool** (or multiple tools) from the dropdown
2. **Choose a timer** — 30s, 60s, or 120s
3. **Type the commands** shown on screen as fast as you can
4. **Get your results** — WPM, accuracy, streak, and rank

## Ranking System

| Rank | WPM Range |
|------|-----------|
| Beginner | 0 - 15 WPM |
| Junior | 15 - 25 WPM |
| Mid-Level | 25 - 35 WPM |
| Senior | 35 - 50 WPM |
| Expert | 50 - 70 WPM |
| Legend | 70+ WPM |

## Why Practice CLI Typing?

- **Interview prep** — DevOps interviews often test your speed on the terminal
- **Productivity** — Faster typing = less time fighting the command line
- **Muscle memory** — Type `kubectl get pods` without thinking
- **Fun challenge** — Compete with yourself and share results
